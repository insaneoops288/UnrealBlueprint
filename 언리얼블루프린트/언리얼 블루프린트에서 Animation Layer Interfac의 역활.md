# **언리얼 엔진 Animation Layer Interface 가이드**

## **1\. Animation Layer Interface란?**

애니메이션 레이어 인터페이스는 애니메이션 블루프린트 간에 공유할 수 있는 \*\*'애니메이션 함수 정의서'\*\*입니다. 일반적인 블루프린트 인터페이스가 변수나 로직을 전달한다면, 이 인터페이스는 \*\*'애니메이션 포즈(Pose)'\*\*를 주고받는 통로를 정의합니다.

## **2\. 주요 역할과 장점**

### **① 모듈화 및 확장성 (Modularity)**

과거에는 하나의 AnimBP 안에 모든 무기(총, 칼, 활 등)에 대한 애니메이션 로직을 다 넣어야 했습니다. 하지만 레이어 인터페이스를 사용하면:

* **Main AnimBP**: 기본 이동 및 공통 로직만 담당.  
* **Weapon AnimBP**: 특정 무기에 특화된 애니메이션 로직만 담당.  
  이후 인터페이스를 통해 필요한 시점에 특정 무기 로직을 Main AnimBP에 '장착(Link)'하여 사용할 수 있습니다.

### **② 협업 효율성 (Collaboration)**

여러 명의 애니메이터나 프로그래머가 작업할 때 유리합니다.

* A 작업자는 캐릭터의 기본 이동 로직을 수정하고,  
* B 작업자는 인터페이스를 상속받은 별도의 파일에서 공격 애니메이션 레이어만 따로 제작할 수 있습니다.  
* 서로 다른 파일을 수정하므로 머지(Merge) 충돌 위험이 줄어듭니다.

### **③ 런타임 교체 (Dynamic Swapping)**

게임 플레이 도중 캐릭터가 무기를 바꾸거나 상태가 변할 때, Set Layer Instance 노드를 사용하여 애니메이션 로직 전체를 실시간으로 교체할 수 있습니다.

## **3\. 작동 원리 및 구성 요소**

### **1\) Animation Layer Interface 애셋**

* 어떤 레이어(함수)가 존재할지 이름을 정의합니다.  
* 각 레이어는 입력 포즈(Input Pose)를 가질 수 있으며, 최종적으로 애니메이션 포즈를 출력합니다.

### **2\) 인터페이스 구현 (Implementation)**

* 애니메이션 블루프린트의 **Class Settings**에서 해당 인터페이스를 추가합니다.  
* AnimGraph 내의 **'Layers'** 탭에서 정의된 레이어의 내부 로직(예: 블렌딩, 시퀀스 재생 등)을 작성합니다.

### **3\) 레이어 연결 (Linking)**

* 메인 AnimBP의 AnimGraph에서 **'Linked Anim Layer'** 노드를 배치합니다.  
* Set Layer Instance 함수를 호출하여 실제 로직이 담긴 다른 AnimBP를 이 노드에 바인딩합니다.

## **4\. 실전 예시: 무기 시스템**

가 가상 시나리오는 다음과 같습니다.

1. **Interface 정의**: I\_WeaponLayers 생성 \-\> UpperBodyLayer 정의.  
2. **Main AnimBP**: UpperBodyLayer 노드를 전신 포즈와 캐시(Cache) 포즈 사이에 배치.  
3. **Rifle AnimBP**: I\_WeaponLayers를 상속받아 UpperBodyLayer 안에서 소총 조준/사격 애니메이션 작성.  
4. **Pistol AnimBP**: 동일 인터페이스를 상속받아 권총용 로직 작성.  
5. **교체**: 캐릭터가 소총을 집으면 Set Layer Instance에 Rifle AnimBP를 넣어 메인 로직에 연결.

## **5\. 요약**

| 구분 | 내용 |
| :---- | :---- |
| **핵심 목적** | 애니메이션 그래프 로직의 분리 및 재사용 |
| **데이터 단위** | 애니메이션 포즈 (Pose) |
| **주요 노드** | Linked Anim Layer, Set Layer Instance |
| **사용 시점** | 프로젝트 규모가 크거나, 무기/장비별로 애니메이션 스타일이 확연히 다를 때 |

이 기능을 활용하면 거대한 하나의 애니메이션 블루프린트가 스파게티처럼 꼬이는 것을 방지하고, 유지보수가 쉬운 깔끔한 구조를 유지할 수 있습니다.
