# **Break Struct Anim Override Sequence 노드 상세 설명**

언리얼 엔진 블루프린트에서 Break Struct Anim Override Sequence 노드는 **Anim Override Sequence**라는 구조체 데이터를 각각의 구성 요소(시퀀스, 재생 속도, 루프 여부 등)로 분해하는 데 사용됩니다.

## **1\. 노드의 주요 역할**

이 노드는 하나의 덩어리로 묶여 있는 '애니메이션 오버라이드' 정보를 해체하여, 그 안에 포함된 세부 설정값들을 블루프린트 로직에서 개별적으로 사용할 수 있도록 꺼내주는 역할을 합니다.

보통 애니메이션 데이터 세트나 데이터 에셋에서 오버라이드 정보를 가져온 뒤, 특정 조건에 따라 애니메이션 시퀀스만 추출하거나 재생 속도를 조절하고 싶을 때 이 노드를 사용합니다.

## **2\. 주요 출력 핀 (Output Pins)**

구조체를 분해하면 일반적으로 다음과 같은 정보들을 얻을 수 있습니다:

* **Sequence (Anim Sequence):** 실제로 재생될 애니메이션 시퀀스 파일입니다.  
* **Play Rate (Float):** 해당 애니메이션이 재생될 속도 배율입니다.  
* **Looping (Boolean):** 애니메이션을 반복 재생할지 여부입니다.  
* **Start Position (Float):** 애니메이션이 시작될 타임라인 상의 위치입니다.

## **3\. 주요 사용 사례**

### **A. 동적 애니메이션 변경**

데이터 테이블(Data Table) 등에 저장된 오버라이드 정보를 불러와서, 현재 캐릭터가 어떤 애니메이션을 재생해야 하는지 확인하고 그 시퀀스 값만 뽑아내어 다른 노드의 입력값으로 전달할 때 사용합니다.

### **B. 로직 분기 생성**

오버라이드 구조체 내의 Looping 값이 true인지 false인지에 따라 블루프린트에서 Branch 노드를 사용해 서로 다른 로직(예: 이펙트 발생, 사운드 재생 등)을 수행하도록 설계할 수 있습니다.

### **C. 디버깅 및 UI 표시**

현재 적용된 오버라이드 애니메이션의 이름이나 재생 속도를 화면(HUD)에 텍스트로 표시하기 위해 구조체 내부의 데이터가 필요할 때 유용합니다.

## **4\. 요약**
