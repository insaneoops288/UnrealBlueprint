# **Break Struct Anim Override 노드 상세 가이드**

언리얼 엔진 블루프린트에서 Break Struct Anim Override 노드는 애니메이션 시스템 내에서 **애니메이션 덮어쓰기(Override)** 정보를 담고 있는 구조체를 분해하여, 그 안에 포함된 세부 설정값들을 개별적으로 제어할 수 있도록 도와주는 역할을 합니다.

## **1\. 노드의 핵심 역할**

이 노드는 Anim Override라는 이름의 구조체 데이터를 입력받아, 그 내부에 저장된 여러 개의 변수들을 각각의 출력 핀으로 나누어 줍니다.

* **데이터 추출:** 묶여 있는 애니메이션 데이터(몽타주, 루프 설정, 재생 속도 등)를 각각 꺼내어 다른 로직에 연결할 수 있게 합니다.  
* **가독성 및 제어:** 전체 구조체를 통째로 넘기는 대신, 필요한 특정 값(예: Play Rate)만 선택하여 로직에 활용할 수 있습니다.

## **2\. 주요 구성 요소 (일반적인 출력 핀)**

사용 중인 플러그인이나 프로젝트의 커스텀 구조체 정의에 따라 차이가 있을 수 있지만, 일반적으로 다음과 같은 정보들을 출력합니다.

| 출력 핀 명칭 | 역할 설명 |
| :---- | :---- |
| **Animation/Montage** | 실행하거나 덮어쓸 실제 애니메이션 시퀀스 또는 몽타주 데이터입니다. |
| **Play Rate** | 애니메이션의 재생 속도를 결정하는 실수(Float) 값입니다. |
| **Looping** | 해당 애니메이션을 반복 재생할지 여부를 결정하는 불리언(Boolean) 값입니다. |
| **Start Time** | 애니메이션의 어느 시점부터 재생을 시작할지 지정하는 값입니다. |
| **Blend In/Out Time** | 애니메이션 전환 시 블렌딩(부드러운 연결)에 소요되는 시간 설정입니다. |

## **3\. 주요 활용 사례**

### **A. 애니메이션 블루프린트(AnimBP)에서의 동적 변경**

캐릭터가 장착한 무기나 현재 상태(부상, 분노 등)에 따라 기본 애니메이션 대신 특정 애니메이션을 덮어써야 할 때 사용합니다. 구조체로 저장된 오버라이드 셋을 분해하여 현재 출력해야 할 애니메이션 노드에 연결합니다.

### **B. 데이터 테이블 연동**

데이터 테이블(Data Table)에 애니메이션 오버라이드 정보를 저장해 두었다가, 필요할 때 Get Data Table Row로 가져온 뒤 이 노드로 분해하여 실제 캐릭터의 애니메이션 로직에 적용합니다.

### **C. 조건부 로직 처리**

예를 들어, 구조체 안의 Play Rate 값만 확인하여 특정 수치 이상일 때만 다른 이펙트를 생성하는 등의 조건문(Branch) 처리에 활용할 수 있습니다.

## **4\. 사용 팁**

* **핀 숨기기 (Hide Unconnected Pins):** 노드를 선택한 후 디테일 패널에서 사용하지 않는 출력 핀들을 체크 해제하면 노드의 크기를 줄여 블루프린트를 깔끔하게 유지할 수 있습니다.  
* **Make Struct와의 쌍:** 반대로 개별 변수들을 하나의 오버라이드 구조체로 묶고 싶을 때는 Make Struct Anim Override 노드를 사용합니다.

**요약하자면:** Break Struct Anim Override는 \*\*"애니메이션 설정 보따리를 풀어서 내용물을 하나씩 꺼내 쓰는 도구"\*\*라고 이해하시면 됩니다.
