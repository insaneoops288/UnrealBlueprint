언리얼 블루프린트에서 **Break Vector 2D** 노드는 하나의 덩어리로 묶여 있는 **Vector 2D 구조체(Struct)를 개별적인 X와 Y 값(Float)으로 분리**해 주는 역할을 합니다.

쉽게 비유하자면, '좌표'라는 이름의 포장된 박스를 뜯어서 그 안에 들어있는 '가로(X)'와 '세로(Y)' 알맹이를 각각 꺼내는 작업이라고 생각하시면 됩니다.

## ---

**1\. 노드의 구조**

* **Input (왼쪽):** **Vector 2D** (X와 Y가 합쳐진 상태의 데이터)  
* **Output (오른쪽):** \* **X:** 벡터의 가로축 값 (단일 실수)  
  * **Y:** 벡터의 세로축 값 (단일 실수)

## ---

**2\. 왜 사용하나요?**

블루프린트에서 많은 데이터(마우스 위치, UI 좌표, 텍스처 UV 등)는 X와 Y가 묶인 **Vector 2D** 형태로 전달됩니다. 하지만 로직을 짜다 보면 **특정 축의 값만 필요**하거나, **각각 따로 계산**해야 할 때가 있습니다.

### **주요 활용 사례:**

* **UI (UMG) 작업:** 위젯의 현재 위치에서 X값(가로 위치)만 가져와서 화면 왼쪽 끝에 있는지 판정할 때.  
* **마우스 입력:** 마우스의 위치 데이터 중 Y값만 사용하여 카메라의 상하 회전을 제어하고 싶을 때.  
* **조건문:** "X 좌표가 500보다 클 때만 실행해라" 같은 로직을 짤 때 (Vector 2D 상태로는 \> 비교 연산을 직접 할 수 없으므로 Break 해서 X만 꺼내야 합니다).  
* **텍스처 좌표 제어:** 머티리얼이나 UI에서 UV 좌표 중 U(X)나 V(Y) 하나만 수정하여 애니메이션을 줄 때.

## ---

**3\. 세트로 쓰이는 노드: Make Vector 2D**

**Break Vector 2D**가 박스를 뜯는 것이라면, 반대로 각각의 X, Y 값을 다시 하나의 Vector 2D 묶음으로 만드는 노드는 **Make Vector 2D**입니다.

* **Break:** Vector 2D → X, Y (분해)  
* **Make:** X, Y → Vector 2D (결합)

## ---

**💡 팁: 노드 없이 바로 분해하기**

노드 그래프가 지저분해지는 게 싫다면, **Break Vector 2D 노드를 직접 꺼내지 않고도** 같은 효과를 낼 수 있는 꿀팁이 있습니다.

1. 분해하고 싶은 **Vector 2D 타입의 변수나 핀** 위에 마우스 오른쪽 버튼을 클릭합니다.  
2. 메뉴에서 \*\*\[구조체 핀 분할 (Split Struct Pin)\]\*\*을 선택합니다.  
3. 그러면 노드가 생기는 대신 해당 핀이 즉시 X와 Y 핀으로 쪼개집니다. (다시 합치고 싶으면 마우스 우클릭 후 **구조체 핀 재결합** 선택)

**요약**

**Break Vector 2D**는 "좌표 뭉치에서 X랑 Y를 따로따로 쓰고 싶을 때" 사용하는 필수적인 데이터 분해 도구입니다.

현재 진행 중인 UI 작업이나 캐릭터 이동 로직에서 특정 좌표값을 제어하는 데 어려움이 있으신가요? 구체적인 상황을 알려주시면 노드 연결 방법을 더 상세히 알려드릴 수 있습니다.
