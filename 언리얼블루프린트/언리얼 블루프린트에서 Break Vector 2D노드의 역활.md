언리얼 블루프린트에서 Break Vector 2D노드의 역활?


언리얼 엔진 블루프린트에서 Break Vector 2D 노드는 하나의 덩어리로 묶여 있는 **Vector 2D(2차원 벡터) 구조체를 각각의 구성 요소인 X와 Y 값으로 분해(Break)**하는 역할을 합니다.

1. 주요 기능
Vector 2D 데이터 타입은 보통 화면 좌표, 마우스 위치, 혹은 2D 평면상의 방향 등을 나타낼 때 사용됩니다. 이 데이터는 기본적으로 (X, Y)가 한 세트로 묶여 있는데, 특정 축의 값만 따로 추출해서 계산이나 로직에 사용하고 싶을 때 이 노드를 사용합니다.

입력: Vector 2D 구조체 (X와 Y가 포함된 하나의 핀)

출력: * X: 벡터의 X축 값 (Float/실수)

Y: 벡터의 Y축 값 (Float/실수)

2. 언제 사용하나요? (활용 예시)
마우스 좌표 활용: 마우스 위치(Get Mouse Position)를 가져온 뒤, 마우스의 X 좌표(가로) 값만 확인해서 캐릭터의 회전을 제어하고 싶을 때 사용합니다.

UI/위젯 배치: 위젯의 위치 값 중 Y(세로) 값만 고정하고 X 값만 실시간으로 변화시키고 싶을 때 유용합니다.

특정 축 비교: 캐릭터의 이동 입력 벡터에서 X 값(좌우)이 0보다 큰지(오른쪽으로 가고 있는지) 등을 개별적으로 체크할 때 사용합니다.

3. 반대되는 노드는?
Break Vector 2D와 정확히 반대되는 기능을 하는 노드는 Make Vector 2D입니다.

Break Vector 2D: 하나를 둘로 나눔 (Vector 2D → X, Y)

Make Vector 2D: 둘을 하나로 합침 (X, Y → Vector 2D)

Tip: 블루프린트 노드에서 Vector 2D 타입의 변수 핀을 마우스 오른쪽 버튼으로 클릭한 뒤 **'구조체 핀 분할(Split Struct Pin)'**을 선택하면, 굳이 Break 노드를 꺼내지 않고도 즉시 X와 Y 핀으로 나누어 볼 수 있습니다. 간단한 작업에서는 이 방법이 훨씬 깔끔합니다!
