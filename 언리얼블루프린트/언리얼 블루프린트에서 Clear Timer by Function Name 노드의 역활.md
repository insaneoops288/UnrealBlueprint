# **Clear Timer by Function Name 노드 가이드**

언리얼 엔진 블루프린트에서 Clear Timer by Function Name 노드는 이전에 **Set Timer by Function Name** 노드를 통해 실행된 타이머를 중단시키고 싶을 때 사용합니다.

## **1\. 주요 역할**

이 노드의 핵심 역할은 \*\*"지정된 오브젝트 내에서 특정 이름을 가진 함수의 타이머 실행을 중지"\*\*하는 것입니다.

* 타이머가 반복(Looping) 중이라면 즉시 멈춥니다.  
* 타이머가 단발성으로 대기 중(Pending)이라면 실행되기 전에 취소됩니다.

## **2\. 노드 핀(Pins) 상세 설명**

| 핀 이름 | 타입 | 설명 |
| :---- | :---- | :---- |
| **Object** | Object Reference | 해당 함수가 포함된 오브젝트입니다. 보통 Self를 연결합니다. |
| **Function Name** | String | 중지하고자 하는 함수의 정확한 이름을 입력합니다. |

## **3\. 작동 원리**

1. 이 노드가 호출되면, 엔진은 Object 핀에 연결된 대상에서 Function Name과 일치하는 타이머가 활성화되어 있는지 확인합니다.  
2. 일치하는 타이머를 찾으면 해당 타이머를 **비활성화**하고 리스트에서 제거합니다.  
3. 이후 해당 함수는 타이머에 의해 다시 호출되지 않습니다.

## **4\. 사용 예시**

* **재생력(HP Regen) 중단**: 캐릭터가 최대 체력에 도달했을 때, 주기적으로 체력을 회복시키던 타이머를 멈춥니다.  
* **쿨다운 취소**: 스킬 준비 중 캐릭터가 상태 이상(스턴 등)에 걸렸을 때 예약된 스킬 발동 타이머를 취소합니다.  
* **발사 루틴 정지**: 총기 발사 버튼을 뗐을 때(On Released), 반복해서 총알을 생성하던 타이머를 중지합니다.

## **5\. 주의사항 및 한계**

* **대소문자 구분**: 함수 이름은 대소문자를 정확히 구분해야 합니다. 이름이 틀리면 아무런 동작도 하지 않습니다.  
* **성능 및 안전성**: 함수 이름을 문자열(String)로 찾기 때문에, 나중에 함수의 이름을 변경하면 이 노드도 수동으로 수정해야 합니다.  
* **대안**: 최근 언리얼 개발에서는 Set Timer by Event를 사용하고, 여기서 반환된 **Timer Handle**을 저장해 두었다가 Clear and Invalidate Timer by Handle 노드로 중지하는 방식을 더 권장합니다. (이 방식이 오타 위험이 없고 더 안전합니다.)

## **6\. 요약**

Clear Timer by Function Name은 복잡한 핸들 관리 없이 **함수 이름만으로 간단하게 타이머를 끄고 싶을 때** 유용한 노드입니다.
