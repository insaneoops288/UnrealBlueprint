# **Detach From Actor 노드 상세 설명**

언리얼 엔진(Unreal Engine) 블루프린트에서 **Detach From Actor** 노드는 특정 액터와 부모 액터 사이의 **계층적 연결(Attachment)을 끊는 역할**을 합니다. 쉽게 말해, Attach To Actor로 붙여놓았던 자식 액터를 부모로부터 독립시키는 기능을 합니다.

## **1\. 주요 역할**

* **부모-자식 관계 해제**: 부모 액터의 움직임이나 회전이 더 이상 자식 액터에게 영향을 주지 않도록 합니다.  
* **물리적 분리**: 캐릭터가 들고 있던 무기를 땅에 떨어뜨리거나, 탑승 중인 탈것에서 내릴 때 주로 사용됩니다.

## **2\. 입력 핀 (Input Pins)**

이 노드에는 분리 시 상태를 결정하는 중요한 설정값들이 있습니다.

### **Detachment Rules (분리 규칙)**

분리되는 순간 액터의 위치, 회전, 크기를 어떻게 처리할지 결정합니다.

1. **Location Rule (위치 규칙)**  
   * **Keep Relative**: 부모를 기준으로 가졌던 상대적 좌표를 유지합니다. (부모가 멀리 있다면 분리 시 갑자기 위치가 바뀔 수 있음)  
   * **Keep World**: 분리되는 순간의 현재 월드 좌표를 그대로 유지합니다. **(가장 많이 사용됨)**  
   * **Snap to Target**: 부모의 원점 위치로 즉시 이동하며 분리됩니다.  
2. **Rotation Rule (회전 규칙)**  
   * **Keep Relative**: 부모 기준 상대적 회전값을 유지합니다.  
   * **Keep World**: 현재 월드상의 회전 상태를 유지합니다.  
   * **Snap to Target**: 부모의 회전값과 일치시킵니다.  
3. **Scale Rule (스케일 규칙)**  
   * **Keep Relative**: 부모 기준 상대적 크기를 유지합니다.  
   * **Keep World**: 현재 월드상의 실제 크기를 유지합니다. (부모의 스케일 영향을 받지 않게 계산됨)

## **3\. 주요 사용 사례**

* **아이템 버리기**: 캐릭터 손(Socket)에 붙어있던 무기 액터를 분리하여 월드에 배치할 때 사용합니다.  
* **탈것 하차**: 탈것 액터의 자식으로 붙어있던 캐릭터를 분리하여 독립적인 이동이 가능하게 할 때 사용합니다.  
* **모듈형 조립**: 로봇이나 기계 장치에서 부품이 파괴되거나 분리되어 나갈 때 사용합니다.

## **4\. 주의사항**

* **Collision (충돌)**: 액터를 부모로부터 분리한 직후에는 충돌 설정이 비활성화되어 있을 수 있습니다. 땅에 떨어지는 연출을 원한다면 분리 후 Set Actor Enable Collision 등을 통해 충돌을 다시 확인해야 합니다.  
* **Physics (물리)**: 분리하자마자 중력의 영향을 받게 하려면, 분리 노드 실행 직후 해당 액터의 메시 컴포넌트에 Set Simulate Physics를 True로 설정해 주어야 합니다.  
* **Target**: Target 핀에는 부모가 아니라 **분리시키고자 하는 자식 액터**를 연결해야 합니다.

## **5\. 유사 노드**

* **Detach From Component**: 액터를 특정 컴포넌트(예: 스켈레탈 메시의 소켓)로부터 분리할 때 사용합니다. 사실상 Detach From Actor와 내부 동작은 거의 유사합니다.
