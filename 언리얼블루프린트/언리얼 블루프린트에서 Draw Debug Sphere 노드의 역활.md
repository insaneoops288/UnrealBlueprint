# **Draw Debug Sphere 노드 역할 및 활용 가이드**

Draw Debug Sphere 노드는 언리얼 엔진에서 **디버깅(Debugging)** 목적으로 가장 빈번하게 사용되는 시각화 도구 중 하나입니다. 게임 실행 중에는 보이지 않아야 할 데이터나 범위를 개발자가 눈으로 직접 확인할 수 있게 해줍니다.

## **1\. 주요 역할 (Main Role)**

이 노드는 3D 공간의 특정 위치에 **와이어프레임 또는 실선 형태의 구체**를 생성합니다. 주된 목적은 다음과 같습니다:

* **위치 확인:** 특정 액터나 컴포넌트의 좌표가 정확한지 시각적으로 표시합니다.  
* **범위 시각화:** 폭발 반경, 캐릭터의 인식 범위, 아이템 획득 가능 거리 등을 확인합니다.  
* **충돌 테스트 결과 표시:** 라인 트레이스(Line Trace)나 스피어 트레이스(Sphere Trace)가 부딪힌 지점(Hit Location)을 표시합니다.  
* **AI 경로 확인:** AI가 이동하려는 목표 지점이나 경로 포인트를 표시합니다.

## **2\. 주요 입력 파라미터 (Parameters)**

노드를 구성하는 핵심 입력값들은 다음과 같습니다:

| 입력 항목 | 타입 | 설명 |
| :---- | :---- | :---- |
| **Center** | Vector | 구체가 그려질 월드 좌표상의 중심점입니다. |
| **Radius** | Float | 구체의 크기(반경)를 결정합니다. |
| **Segments** | Integer | 구체를 구성하는 선의 개수입니다. 값이 높을수록 부드러운 구가 되지만 성능을 더 사용합니다. (보통 12\~24 적당) |
| **Line Color** | Linear Color | 구체 선의 색상을 정합니다. (예: 위험 지역은 빨간색, 안전 지역은 초록색) |
| **Duration** | Float | 구체가 화면에 유지될 시간(초)입니다. 0을 입력하면 해당 프레임에만 나타납니다. |
| **Thickness** | Float | 구체를 그리는 선의 굵기입니다. |

## **3\. 실무 활용 예시 (Practical Examples)**

### **① 폭발 반경 확인 (Explosion Radius)**

수류탄이나 폭발물이 터질 때, 실제로 대미지가 들어가는 범위를 시각적으로 확인하고 싶을 때 사용합니다.

* **Center:** 폭발 위치  
* **Radius:** 폭발 대미지 반경 값

### **② 캐릭터 인식 범위 (AI Perception)**

AI 적 캐릭터가 플레이어를 감지할 수 있는 거리를 디버그 모드에서 확인하여 밸런스를 조절할 때 유용합니다.

* **Center:** AI 캐릭터의 위치  
* **Radius:** 인식 가능 거리

### **③ 히트 포인트 마킹 (Hit Point Marking)**

총기 발사 시스템에서 총알이 벽이나 물체에 맞은 정확한 위치를 점 찍듯 표현할 수 있습니다.

* **Center:** 트레이스 결과의 Hit Location  
* **Radius:** 5.0 \~ 10.0 정도의 작은 값

## **4\. 주의사항 (Tips & Notes)**

1. **배포 시 자동 제거:** 일반적으로 "Debug"가 붙은 노드들은 게임을 최종 빌드(Shipping)할 때 화면에 나타나지 않도록 설계되어 있지만, 성능 최적화를 위해 개발 완료 후에는 노드를 끊어두거나 Is Debug Mode 같은 불리언 변수로 제어하는 것이 좋습니다.  
2. **성능 영향:** 매 프레임(Tick)마다 너무 많은 Draw Debug Sphere를 호출하면 프레임 드랍이 발생할 수 있으니 주의하세요.  
3. **지속 시간 활용:** Duration을 \-1로 설정하면 명령으로 지우기 전까지 영구적으로 표시됩니다. (일시적인 테스트 시에는 보통 2\~5초가 적당합니다.)

## **5\. 유사한 노드들**

* **Draw Debug Box:** 상자 형태의 범위 표시  
* **Draw Debug Line:** 두 지점 사이의 선 표시  
* **Draw Debug Capsule:** 캐릭터 캡슐 컴포넌트 형태 표시  
* **Draw Debug Arrow:** 방향성(Vector) 표시
