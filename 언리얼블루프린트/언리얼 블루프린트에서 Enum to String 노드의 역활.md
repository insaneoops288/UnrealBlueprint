# **Enum to String 노드 가이드**

언리얼 엔진 블루프린트에서 Enum to String 노드는 특정 열거형(Enum) 값을 대응하는 이름의 문자열로 변환하는 데 사용됩니다.

## **1\. 주요 역할**

### **① 디버깅 (Debugging)**

가장 많이 사용되는 용도입니다. 캐릭터의 상태(예: Idle, Running, Jumping)가 열거형으로 정의되어 있을 때, Print String 노드와 연결하여 현재 캐릭터가 어떤 상태인지 화면에 실시간으로 텍스트로 찍어볼 수 있습니다.

### **② UI 표시**

사용자 인터페이스(UMG)에서 특정 선택 사항이나 상태를 텍스트로 보여줄 때 유용합니다. 예를 들어, 게임 설정의 '그래픽 품질(Low, Medium, High)' 열거형 값을 그대로 텍스트 블록에 표시할 수 있습니다.

### **③ 로그 및 데이터 저장**

시스템 로그에 현재 상태를 기록하거나, 특정 상태 이름을 텍스트 형식으로 저장해야 할 때 사용합니다.

## **2\. 사용 방법**

1. 블루프린트 그래프에서 열거형 변수를 가져옵니다(Get).  
2. 변수의 출력 핀을 드래그하여 빈 공간에 놓습니다.  
3. 검색창에 Enum to String을 입력하거나, 단순히 To String을 검색하여 해당 열거형 타입에 맞는 변환 노드를 선택합니다.  
4. 반환된 **String** 값을 Print String이나 UI 텍스트 변수에 연결합니다.

## **3\. 특징 및 주의사항**

### **내부 이름 출력**

Enum to String은 기본적으로 열거형 정의 시 입력한 \*\*내용(Internal Name)\*\*을 출력합니다.

* 예: VE\_Walking이라고 정의했다면 문자열로 "VE\_Walking"이 나옵니다.

### **Display Name과의 차이**

언리얼 에디터에서 설정한 '표시 이름(Display Name)'과는 다를 수 있습니다. 만약 사용자에게 보여줄 예쁜 이름을 원한다면 Get Enumerator Display Name 노드를 사용하는 것이 더 적합합니다.

* **Enum to String**: 프로그래밍적인 식별자 이름을 반환.  
* **Get Enumerator Display Name**: 에디터에서 별도로 설정한 친숙한 이름을 반환(로컬라이징 가능).

### **자동 형변환 (Autocast)**

최신 언리얼 엔진 버전에서는 열거형 출력 핀을 Print String의 입력 핀(String 타입)에 직접 연결하면, 엔진이 자동으로 Enum to String 노드를 생성하여 연결해 줍니다.

## **4\. 실전 예시**

**상태 출력 시나리오:**

* **Enum 타입**: E\_PlayerState (Idle, Attack, Dead)  
* **로직**: 플레이어가 대미지를 입을 때마다 현재 상태를 화면 왼쪽 상단에 표시.  
* **구성**: E\_PlayerState 변수 \-\> Enum to String \-\> Print String

### **요약**

Enum to String은 \*\*"컴퓨터가 사용하는 숫자 형태의 열거형 값을 사람이 읽을 수 있는 텍스트로 바꿔주는 통번역기"\*\*라고 이해하시면 됩니다. 주로 개발 중 상태를 확인하거나 간단한 UI 메시지를 만들 때 필수적인 노드입니다.
