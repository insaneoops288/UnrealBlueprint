# **Foot Placement 노드의 역할과 기능**

언리얼 엔진(특히 UE5 이상)에서 제공되는 **Foot Placement** 노드는 캐릭터의 애니메이션이 재생될 때, 발이 공중에 떠 있거나 땅속으로 파고드는 현상을 방지하여 물리적으로 자연스러운 움직임을 구현하는 핵심 노드입니다.

## **1\. 주요 역할 (Primary Roles)**

### **발의 위치 조정 (Positioning)**

캐릭터가 평지가 아닌 경사로나 계단을 걸을 때, 원래의 애니메이션은 평지 기준이기 때문에 발이 공중에 뜨거나 지면을 뚫고 들어갈 수 있습니다. Foot Placement 노드는 실시간으로 지면의 높이를 감지하여 발의 위치를 정확히 지면 위로 옮깁니다.

### **발의 회전 조정 (Alignment)**

단순히 높이만 맞추는 것이 아니라, 지면의 노멀(Normal, 법선) 값을 계산하여 발바닥이 지면의 기울기에 딱 붙도록 회전시킵니다.

### **골반 높이 조정 (Pelvis/Hip Adjustment)**

한쪽 발이 높은 곳을 딛게 되면 몸 전체의 균형을 위해 골반(Pelvis) 위치도 자연스럽게 낮아져야 합니다. Foot Placement 노드는 다리 길이를 고려하여 골반의 높이를 자동으로 보정합니다.

## **2\. 작동 원리 (How it Works)**

Foot Placement 노드는 내부적으로 다음과 같은 과정을 거칩니다.

1. **트레이스(Trace):** 각 발의 위치에서 아래 방향으로 가상의 선(Raycast)을 쏘아 지면과의 거리를 측정합니다.  
2. **오프셋 계산:** 원래 애니메이션상의 발 위치와 실제 지면 사이의 차이(Offset)를 계산합니다.  
3. **IK 체인 적용:** 계산된 오프셋을 바탕으로 다리 관절(허벅지-종아리-발목)의 각도를 역계산하여 발을 목표 지점에 위치시킵니다.  
4. **잠김(Locking) 및 보간:** 발이 지면에 닿아 있는 동안 미끄러지지 않도록 고정하거나, 변화가 급격할 때 부드럽게 전환되도록 보간(Interpolation) 처리를 수행합니다.

## **3\. 주요 속성 (Key Properties)**

| 속성 | 설명 |
| :---- | :---- |
| **Foot Components** | IK를 적용할 발 뼈(Bone)와 관련된 데이터를 정의합니다. |
| **Pelvis Bone** | 높이 조정을 위해 기준이 될 골반 뼈를 지정합니다. |
| **Trace Distance** | 지면을 감지하기 위해 발 위아래로 얼마나 멀리 검사할지 결정합니다. |
| **Interpolation Speed** | 발이 지면에 닿을 때 얼마나 빠르게 반응할지 설정합니다. (너무 낮으면 느리게 반응하고, 너무 높으면 떨림 현상이 발생할 수 있음) |

## **4\. 사용 시 장점**

* **몰입감 향상:** 캐릭터가 환경과 상호작용하고 있다는 느낌을 강하게 줍니다.  
* **개발 효율성:** 과거에는 복잡한 수학 연산과 블루프린트 로직으로 직접 IK 시스템을 구축해야 했으나, 이제는 노드 하나로 고품질의 결과를 얻을 수 있습니다.  
* **자동화:** 다양한 지형 환경에 일일이 대응할 필요 없이 시스템이 알아서 대응합니다.

## **5\. 구현 팁**

* **애니메이션 그래프(AnimGraph):** 이 노드는 보통 애니메이션 블루프린트의 AnimGraph 마지막 단계(로컬 또는 컴포넌트 공간) 근처에 배치됩니다.  
* **가상 뼈(Virtual Bones):** 더 정밀한 제어를 위해 가상 뼈를 트레이스 기준으로 사용하는 경우가 많습니다.  
* **Control Rig 통합:** 최근 UE5에서는 이 기능을 Control Rig 내에서 더욱 세밀하게 제어하거나, 전용 Foot Placement 레이어를 통해 더 복잡한 물리 상호작용을 처리하기도 합니다.
