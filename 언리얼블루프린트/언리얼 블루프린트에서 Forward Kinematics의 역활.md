# **언리얼 엔진 순운동학 (Forward Kinematics, FK) 가이드**

순운동학(FK)은 애니메이션과 로보틱스에서 사용하는 방식으로, 상위 부모 관절의 회전과 위치가 하위 자식 관절에 영향을 주는 **계층적 이동 방식**을 의미합니다.

## **1\. FK의 핵심 개념**

FK는 "부모에서 자식으로" 데이터가 흐릅니다.

* **계층 구조:** 어깨 → 팔꿈치 → 손목 → 손가락 순으로 움직임이 전달됩니다.  
* **결정 요소:** 각 관절의 \*\*회전 값(Rotation)\*\*이 최종적인 끝점(End Effector, 예: 손 끝)의 위치를 결정합니다.  
* **연산:** 부모 관절의 트랜스폼(Transform)에 자식의 로컬 트랜스폼을 곱하여 월드 좌표를 계산합니다.

## **2\. 언리얼 엔진 블루프린트에서의 주요 역할**

### **① 애니메이션 데이터의 기본 재생**

우리가 흔히 사용하는 .fbx 애니메이션 파일은 대부분 FK 기반입니다. 각 프레임마다 본(Bone)들의 회전 값이 저장되어 있으며, 언리얼은 이를 순차적으로 계산해 캐릭터의 포즈를 구현합니다.

### **② 컨트롤 릭(Control Rig)에서의 정밀 제어**

컨트롤 릭 내에서 FK 노드를 사용하면 애니메이터가 수동으로 관절을 하나하나 조정할 수 있습니다.

* 특정 관절에 오프셋(Offset)을 주어 기존 애니메이션을 보정할 때 사용합니다.  
* 가장 직관적이고 예측 가능한 움직임을 만들어냅니다.

### **③ 소켓(Socket) 및 부착(Attachment) 시스템**

블루프린트에서 GetSocketTransform이나 AttachToComponent를 사용할 때 FK 원리가 적용됩니다. 부모 본이 움직이면 그에 속한 소켓과 부착된 아이템(무기 등)이 자동으로 따라가는 것은 FK 계층 구조 덕분입니다.

### **④ 절차적 애니메이션 (Procedural Animation)**

블루프린트 노드(Modify Bone 등)를 통해 실시간으로 본의 회전을 제어할 때 주로 FK를 사용합니다.

* 예: 캐릭터가 마우스 방향으로 고개를 돌리는 기능 (Look At).  
* 예: 차량의 바퀴가 속도에 맞춰 회전하는 기능.

## **3\. FK vs IK (역운동학) 비교**

| 특징 | 순운동학 (FK) | 역운동학 (IK) |
| :---- | :---- | :---- |
| **데이터 흐름** | 부모 → 자식 (Top-down) | 자식(끝점) → 부모 (Bottom-up) |
| **제어 대상** | 관절의 회전값 (Rotation) | 끝점의 목표 위치 (Target Position) |
| **장점** | 계산이 가볍고 결과가 예측 가능함 | 특정 지점을 정확히 짚거나 밟기 좋음 |
| **단점** | 손이나 발을 특정 위치에 두기 어려움 | 계산량이 많고 관절 꺾임이 부자연스러울 수 있음 |
| **주요 용어** | 회전 호(Arc), 계층 구조 | 이펙터(Effector), 폴 벡터(Pole Vector) |

## **4\. 블루프린트 활용 예시**

블루프린트에서 FK를 활용하여 캐릭터의 팔을 위로 드는 로직을 짤 때의 흐름은 다음과 같습니다.

1. **본 수정 노드 사용:** 애니메이션 블루프린트(AnimBP)의 애님 그래프에서 Modify Bone 노드를 배치합니다.  
2. **본 선택:** UpperArm\_L (왼쪽 상완) 본을 선택합니다.  
3. **회전 설정:** Rotation 핀에 변수를 연결하고, Rotation Mode를 Add to Existing으로 설정합니다.  
4. **결과:** 해당 변수 값이 커질수록 어깨 관절이 회전하며, 팔꿈치와 손목은 별도의 연산 없이도 어깨를 따라 함께 들어 올려집니다.

## **5\. 결론: 언제 FK를 사용해야 하는가?**

* **자유로운 움직임:** 공중에서 팔을 휘두르거나 꼬리를 흔드는 등 끝점이 무언가에 닿을 필요가 없을 때.  
* **성능 최적화:** 많은 수의 캐릭터가 등장하여 복잡한 IK 연산이 부담스러울 때.  
* **전통적인 애니메이션:** 이미 제작된 고품질의 모션 캡처 데이터를 활용할 때.

FK는 언리얼 엔진 애니메이션의 **뿌리**와 같으며, 이를 기반으로 IK와 같은 고급 기술이 더해져 더욱 사실적인 움직임이 완성됩니다.
