# **Get Actor Location 노드 가이드**

언리얼 엔진 블루프린트에서 **Get Actor Location** 노드는 월드 내에 존재하는 특정 액터(Actor)의 현재 \*\*3차원 좌표(X, Y, Z)\*\*를 가져오는 역할을 합니다.

## **1\. 기본 개념**

이 노드는 지정된 액터의 'Root Component'(루트 컴포넌트)를 기준으로 월드 공간(World Space)에서의 위치를 **Vector(벡터)** 데이터 타입으로 반환합니다.

* **반환 값 (Return Value):** 액터의 X, Y, Z 좌표를 포함하는 벡터 데이터.  
* **대상 (Target):** 위치를 확인하려는 액터입니다. (기본값은 'Self', 즉 노드가 포함된 블루프린트 자신)

## **2\. 주요 특징**

* **월드 좌표 기준:** 로컬 좌표(Relative Location)가 아닌, 게임 월드 전체의 절대적인 0,0,0 좌표를 기준으로 한 위치를 알려줍니다.  
* **실시간 데이터:** 노드가 실행되는 시점의 최신 좌표를 반환하므로, 이동 중인 캐릭터의 위치를 매 프레임마다 확인하기에 적합합니다.

## **3\. 노드 구성**

1. **Target (입력 핀):** 위치를 알고 싶은 액터의 레퍼런스를 연결합니다. 연결하지 않으면 블루프린트 자신(Self)의 위치를 가져옵니다.  
2. **Return Value (출력 핀):** 액터의 현재 위치가 담긴 노란색 벡터 핀입니다. 이 핀을 우클릭하여 '구조체 핀 분할(Split Struct Pin)'을 선택하면 X, Y, Z 값을 각각 따로 사용할 수 있습니다.

## **4\. 실전 활용 사례**

### **① 두 대상 사이의 거리 계산**

Get Actor Location으로 내 위치를 가져오고, 다른 액터의 위치를 가져와 Distance (Vector) 노드에 연결하면 두 대상 사이의 거리를 쉽게 계산할 수 있습니다. 이를 통해 적이 사거리 안에 있는지 체크할 수 있습니다.

### **② 아이템 생성 (Spawn Actor)**

특정 몬스터가 죽었을 때 그 자리에 아이템을 소환하고 싶다면, 죽는 시점에 몬스터의 Get Actor Location 값을 가져와 Spawn Actor from Class 노드의 Spawn Transform 위치 값으로 넣어줍니다.

### **③ 특정 높이 체크**

캐릭터가 낭떠러지로 떨어졌는지 확인하고 싶을 때, Return Value의 **Z**값(높이)만 추출하여 특정 수치(예: \-1000)보다 작아졌는지 비교하는 로직을 짤 수 있습니다.

### **④ 발사체 발사 위치 지정**

플레이어가 총을 쏠 때 총구 위치나 캐릭터의 현재 위치에서 탄환이 생성되도록 설정할 때 사용합니다.

## **5\. 주의사항**

* **Root Component 확인:** 액터의 루트 컴포넌트 위치를 기준으로 하기 때문에, 만약 액터 내부의 메시(Mesh)가 루트로부터 멀리 떨어져 있다면 시각적인 위치와 반환되는 좌표값이 다를 수 있습니다.  
* **Get World Location과의 차이:** Get Actor Location은 액터 전체의 기준점을 가져오고, Get World Location은 액터 내의 특정 컴포넌트(예: 손, 카메라, 센서)의 위치를 가져올 때 사용합니다.
