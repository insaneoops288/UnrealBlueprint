# **Get Actor Transform 노드 완벽 이해하기**

언리얼 엔진 블루프린트에서 Get Actor Transform 노드는 액터의 **Transform(트랜스폼)** 정보를 추출하는 데 사용됩니다. 트랜스폼은 3D 공간에서 물체의 상태를 정의하는 세 가지 핵심 데이터를 포함하는 구조체(Struct)입니다.

## **1\. 트랜스폼의 3대 구성 요소**

이 노드는 결과값으로 Transform 구조체를 반환하며, 이를 분할(Break)하거나 그대로 사용할 수 있습니다.

* **Location (위치):** 액터가 월드 공간의 어디에 있는지 나타내는 Vector (X, Y, Z) 값입니다.  
* **Rotation (회전):** 액터가 어느 방향을 보고 있는지 나타내는 Rotator (Pitch, Roll, Yaw) 값입니다.  
* **Scale (스케일):** 액터의 크기가 기본값 대비 얼마나 크거나 작은지 나타내는 Vector (X, Y, Z) 값입니다.

## **2\. 노드의 주요 특징**

* **World Space 기준:** 기본적으로 이 노드는 **월드 좌표계**를 기준으로 값을 가져옵니다. 즉, 부모 액터가 있더라도 부모 기준의 상대적 좌표가 아닌, 전체 게임 세상에서의 절대적인 위치와 회전값을 반환합니다.  
* **Target (대상):** 기본값은 Self로 설정되어 있어, 노드를 배치한 해당 블루프린트 액터의 정보를 가져옵니다. 다른 액터의 변수나 레퍼런스를 연결하면 해당 액터의 정보를 가져올 수도 있습니다.

## **3\. 주요 활용 사례**

### **① 다른 액터 스폰 (Spawn Actor from Class)**

새로운 액터(예: 총알, 효과)를 생성할 때, 생성 위치와 방향을 현재 캐릭터의 위치와 방향에 맞추기 위해 Get Actor Transform 결과를 Spawn Actor 노드의 Spawn Transform 핀에 바로 연결합니다.

### **② 액터 위치 및 상태 저장**

체크포인트 시스템이나 세이브 기능을 구현할 때, 플레이어의 현재 위치와 상태를 한 번에 저장하기 위해 이 정보를 추출합니다.

### **③ 트랜스폼 계산 및 보간**

특정 지점으로 부드럽게 이동하거나 회전시키고 싶을 때, 시작점 정보를 얻기 위해 사용합니다. Lerp (Transform) 노드와 함께 자주 사용됩니다.

## **4\. 관련 노드 비교**

| 노드 이름 | 반환 데이터 | 용도 |
| :---- | :---- | :---- |
| **Get Actor Location** | Vector | 위치 정보만 필요할 때 |
| **Get Actor Rotation** | Rotator | 회전 정보만 필요할 때 |
| **Get Actor Scale 3D** | Vector | 크기 정보만 필요할 때 |
| **Get Actor Transform** | **Transform** | **위의 세 가지가 모두 필요하거나, 다른 노드의 입력값으로 트랜스폼이 요구될 때** |

## **5\. 팁: 구조체 분할하기 (Break Transform)**

Get Actor Transform의 반환값 핀에서 마우스 오른쪽 버튼을 클릭하고 \*\*'Split Struct Pin'\*\*을 선택하면, 위치/회전/스케일 핀이 각각 나타나서 필요한 값만 골라 쓸 수 있습니다. 반대로 Make Transform을 통해 이 값들을 다시 하나로 합칠 수도 있습니다.
