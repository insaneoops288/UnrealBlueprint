# **언리얼 엔진: Get Actor Up Vector 노드 완벽 가이드**

Get Actor Up Vector 노드는 언리얼 엔진 블루프린트에서 액터의 \*\*로컬 Z축(상단 방향)\*\*이 세계 공간(World Space)에서 어디를 향하고 있는지를 나타내는 \*\*단위 벡터(Unit Vector)\*\*를 반환하는 함수입니다.

## **1\. 핵심 개념**

### **로컬 축과 월드 공간의 변환**

* **로컬 공간(Local Space):** 모든 액터는 자신만의 앞(X), 오른쪽(Y), 위(Z) 축을 가집니다. 액터가 회전하더라도 액터 입장에서 '위'는 항상 로컬 Z축입니다.  
* **월드 공간(World Space):** 게임 전체의 절대적인 좌표계입니다.  
* **노드의 역할:** 액터가 90도 기울어져 있다면, 이 노드는 월드 좌표 기준으로 (0, 0, 1)이 아닌, 기울어진 방향의 벡터를 계산해서 알려줍니다.

### **벡터의 특징**

* **길이(Magnitude):** 항상 **1.0**입니다 (단위 벡터).  
* **방향:** 액터의 정수리(상단) 방향을 가리킵니다.

## **2\. 주요 활용 사례**

### **1\) 액터 상단에 컴포넌트 배치**

액터의 머리 위에 텍스트 렌더러나 아이템 아이콘을 띄우고 싶을 때 사용합니다.

* **공식:** Get Actor Location \+ (Get Actor Up Vector \* 거리)  
* 이렇게 하면 액터가 옆으로 누워있더라도 항상 액터의 '머리 쪽'에 아이템이 배치됩니다.

### **2\) 발사체나 물리 힘 적용**

캐릭터가 점프하거나 물체를 위로 밀어낼 때, 단순히 월드 Z(0,0,1) 방향이 아니라 캐릭터가 서 있는 기준에서의 '위'로 힘을 주고 싶을 때 사용합니다.

* Add Impulse 노드에 이 벡터를 연결하여 방향을 결정합니다.

### **3\) 표면 정렬 (Surface Alignment)**

캐릭터가 경사면을 걸어갈 때, 캐릭터의 몸을 경사면의 수직 방향(Normal)에 맞게 회전시키고 싶을 때 Up Vector를 참조하여 회전값을 계산합니다.

## **3\. 다른 벡터 노드와의 비교**

| 노드 이름 | 방향 (로컬 축) | 주로 쓰이는 곳 |
| :---- | :---- | :---- |
| **Get Actor Forward Vector** | X축 (앞) | 이동 방향, 총알 발사 방향 |
| **Get Actor Right Vector** | Y축 (오른쪽) | 횡이동(Strafe), 좌우 회전 계산 |
| **Get Actor Up Vector** | Z축 (위) | 부양, 머리 위 UI, 중력 반대 방향 |

## **4\. 블루프린트 예시 로직**

**\[액터의 머리 위 200유닛 지점에 파티클 생성하기\]**

1. Get Actor Location으로 현재 위치를 가져옵니다.  
2. Get Actor Up Vector를 가져옵니다.  
3. Up Vector에 Float \* Vector 노드를 사용하여 200을 곱합니다.  
4. 1번의 위치와 3번의 결과값을 더합니다(Vector \+ Vector).  
5. 최종 위치를 Spawn Emitter at Location의 위치 핀에 연결합니다.

## **5\. 주의사항**

* 이 노드는 액터의 \*\*회전(Rotation)\*\*값에 의존합니다. 만약 액터의 Static Mesh가 로컬 좌표계에서 잘못 정렬되어 있다면(예: 모델링 단계에서 Z축이 옆을 향함), Get Actor Up Vector도 의도치 않은 방향을 가리킬 수 있습니다.  
* 단순히 "하늘" 방향이 필요하다면 이 노드 대신 Vector(0, 0, 1)을 직접 사용하는 것이 좋습니다. 이 노드는 **액터가 기울어짐에 따라 같이 변하는 위쪽**이 필요할 때만 사용하세요.
