언리얼 블루프린트에서 Get Actor Up Vector노드의 역활?

언리얼 엔진 블루프린트에서 Get Actor Up Vector 노드는 특정 액터가 바라보고 있는 방향을 기준으로 **'위쪽(상단)'**이 어디인지를 나타내는 방향 데이터를 가져오는 역할을 합니다.

단순히 세상의 위(하늘 방향)를 알려주는 것이 아니라, 액터의 현재 회전 값(Rotation)을 반영한 상대적인 위쪽을 알려준다는 점이 핵심입니다.

1. 주요 특징
단위 벡터(Unit Vector): 노드의 출력값은 항상 길이가 1.0인 정규화된 벡터값입니다.

로컬 Z축 기반: 액터의 로컬 좌표계에서 Z축 방향을 월드 좌표계 기준으로 변환해서 반환합니다.

회전 대응: 액터가 옆으로 기울어지면 'Up Vector'도 함께 기울어집니다. 예를 들어, 사람이 바닥에 누워 있다면 Up Vector는 하늘이 아닌 옆을 향하게 됩니다.

2. 언제 사용하나요? (활용 예시)
이 노드는 주로 액터의 방향성을 제어하거나 물리적인 계산을 할 때 자주 사용됩니다.

점프 및 발사: 캐릭터가 서 있는 방향을 기준으로 수직 점프를 시키거나, 탱크 포탑 위쪽으로 연기 파티클을 생성할 때 사용합니다.

표면 정렬: 캐릭터가 가파른 경사면에 서 있을 때, 발바닥이 지면과 평행하게 닿도록 액터의 방향을 수정할 때 활용합니다.

중력 방향 변경: 특정 액터에게만 적용되는 중력 방향을 설정하거나, 벽을 타고 걷는 로직을 구현할 때 '어디가 위쪽인지' 판단하는 기준으로 사용합니다.

벡터 연산: Get Actor Forward Vector(앞)나 Get Actor Right Vector(오른쪽)와 조합하여 특정 3D 공간 내의 좌표를 계산할 때 필수적입니다.

3. 다른 벡터 노드와의 비교
노드 이름	방향	로컬 축
Get Actor Forward Vector	앞방향	X축
Get Actor Right Vector	오른쪽 방향	Y축
Get Actor Up Vector	윗방향	Z축
요약하자면
Get Actor Up Vector는 "지금 이 물체의 정수리(머리 꼭대기)가 월드 공간에서 어느 방향을 가리키고 있는가?"를 알려주는 나침반과 같습니다. 액터가 회전함에 따라 이 값도 실시간으로 변하므로, 동적인 움직임을 구현할 때 매우 유용합니다.
