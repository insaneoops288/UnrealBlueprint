# **Get Curve Value 노드 가이드**

언리얼 엔진 블루프린트에서 Get Curve Value 노드는 수학적인 계산이나 복잡한 로직 없이도, 시각적으로 편집된 그래프 데이터를 실시간으로 가져와 게임 로직에 반영할 때 필수적으로 사용됩니다.

## **1\. 노드의 주요 역할**

이 노드는 특정 **커브 에셋**을 참조하여, 입력값(InTime)에 대응하는 결과값(Return Value)을 반환합니다.

* **입력 (InTime):** 보통 시간을 의미하지만, 0에서 1 사이의 비율이나 특정 진행 상태를 나타내는 실수(Float) 값을 넣습니다.  
* **출력 (Return Value):** 해당 시점의 커브 그래프 높이(Y축 값)를 반환합니다.

## **2\. 사용 가능한 커브 에셋 종류**

Get Curve Value는 참조하는 에셋의 타입에 따라 결과값이 달라집니다.

| 커브 타입 | 설명 | 용도 |
| :---- | :---- | :---- |
| **Curve Float** | 단일 실수값을 반환 | 투명도 조절, 이동 속도 변화, 크기 조절 등 |
| **Curve Vector** | X, Y, Z 세 가지 실수값을 반환 | 3D 공간 상의 복잡한 경로 이동, 반동 패턴 등 |
| **Curve Linear Color** | RGBA 색상 값을 반환 | 시간에 따른 조명 색상 변화, 머티리얼 색상 변경 등 |

## **3\. 주요 활용 사례**

### **① 애니메이션 및 보간 (Interpolation)**

타임라인(Timeline)과 함께 사용하거나, Delta Seconds를 누적한 변수를 InTime에 연결하여 부드러운 움직임을 구현합니다.

* **예:** 문이 서서히 열릴 때 처음에는 빠르게, 끝에서는 천천히 멈추는(Ease-out) 효과.

### **② 게임 플레이 밸런싱**

수식을 코드로 짜는 대신 커브 에셋에서 그래프를 그려 캐릭터의 레벨별 경험치 요구량이나 거리에 따른 대미지 감쇠(Falloff)를 설정합니다.

* **예:** 타겟과의 거리를 InTime으로 넣고, 그에 따른 대미지 배율을 Return Value로 받음.

### **③ 비주얼 이펙트 (VFX)**

머티리얼 파라미터나 라이트의 강도를 시간에 따라 동적으로 변화시킬 때 사용합니다.

## **4\. 블루프린트 구성 예시**

1. **변수 생성:** Curve Float 타입의 변수를 생성하고 에셋을 할당합니다.  
2. **노드 배치:** 변수를 Get으로 가져온 뒤 선을 끌어 Get Curve Value를 검색합니다.  
3. **값 입력:** InTime에 현재 시간이나 진행도(Alpha)를 연결합니다.  
4. **결과 사용:** 출력된 값을 Set Actor Scale 3D나 Set Vector Parameter Value 등에 연결합니다.

## **5\. 주의사항**

* **에셋 참조 필수:** 노드 자체만으로는 작동하지 않으며, 반드시 어떤 커브 에셋을 읽을지 Target에 지정해줘야 합니다.  
* **범위 확인:** 커브 에셋의 가로축(Time) 범위를 확인하세요. 만약 커브는 0\~1초까지인데 2초를 입력하면, 일반적으로 커브의 마지막 값이 계속 출력됩니다 (설정에 따라 루프 가능).
