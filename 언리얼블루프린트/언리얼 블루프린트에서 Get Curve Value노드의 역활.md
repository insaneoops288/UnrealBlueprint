언리얼 블루프린트에서 Get Curve Value노드의 역활?

언리얼 엔진 블루프린트에서 Get Curve Value 노드는 미리 정의된 커브(Curve) 에셋으로부터 특정 시간(X축)에 해당하는 값(Y축)을 추출할 때 사용합니다.

쉽게 말해, 복잡한 수학 공식 없이도 **"시간에 따라 변화하는 값의 흐름"**을 데이터화해서 가져오는 역할을 합니다.

1. 주요 기능
이 노드는 주로 CurveFloat(플로트 커브) 에셋과 함께 사용되며, 다음과 같은 구조로 작동합니다.

입력(In):

Curve Asset: 값을 가져올 커브 데이터 에셋 (Float, Vector, Linear Color 등).

In Time: 커브 그래프상의 가로축(X축) 위치입니다. 보통 초 단위의 시간을 넣습니다.

출력(Out):

Return Value: 해당 시간 지점의 세로축(Y축) 값입니다.

2. 언제 사용하나요?
단순한 선형 이동이 아니라, 그래프를 그리듯 정교하게 변화하는 값이 필요할 때 필수적입니다.

부드러운 애니메이션 (Custom Lerp):
단순히 A에서 B로 이동하는 것이 아니라, 처음엔 느리다가 중간에 빨라지고 끝에서 다시 느려지는(Ease In/Out) 움직임을 구현할 때 사용합니다.

절차적 움직임:
캐릭터가 점프할 때의 궤적, 혹은 카메라 흔들림(Camera Shake)의 강도를 커브로 그려놓고 매 프레임마다 시간을 입력해 값을 받아올 수 있습니다.

비주얼 이펙트(VFX) 제어:
시간에 따른 머티리얼의 투명도 변화나 파티클의 크기 조절 등을 블루프린트에서 제어할 때 유용합니다.

3. 타임라인(Timeline) 노드와의 차이점
특징	Get Curve Value	타임라인 (Timeline)
작동 방식	단순히 특정 시점의 값을 한 번 호출함	시간에 따라 자동으로 이벤트를 발생시키며 값을 갱신함
흐름 제어	실행 핀(Exec)이 필요할 때마다 직접 호출	자체적으로 재생(Play), 정지(Stop) 기능 포함
유연성	루프문이나 복잡한 계산식 안에서 값만 뽑아 쓰기 좋음	일정한 시간 동안 벌어지는 연속적인 동작에 적합함
4. 사용 팁
커브 에셋은 콘텐츠 브라우저에서 우클릭 -> Miscellaneous(기타) -> Curve를 선택하여 생성할 수 있습니다.

Get Curve Value 노드는 에셋을 직접 참조하므로, 여러 블루프린트에서 동일한 커브 데이터를 공유해서 사용할 때 매우 효율적입니다.
