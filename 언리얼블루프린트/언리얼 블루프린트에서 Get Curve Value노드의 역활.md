언리얼 블루프린트에서 **Get Curve Value** 노드는 미리 정의된 **커브(Curve) 데이터에서 특정 시점의 값(Float)을 추출**하는 역할을 합니다.

단순히 일직선으로 변하는 값이 아니라, 곡선 형태의 복잡한 변화를 비선형적으로 가져오고 싶을 때 필수적으로 사용됩니다. 이 노드는 크게 두 가지 맥락(애니메이션과 일반 로직)에서 사용됩니다.

## ---

**1\. 두 가지 주요 사용 맥락**

### **① 애니메이션 블루프린트 (AnimBP)에서의 역할**

가장 많이 쓰이는 형태입니다. 애니메이션 시퀀스 내부에 만든 \*\*'애니메이션 커브'\*\*의 현재 값을 실시간으로 가져옵니다.

* **용도:** 애니메이션 재생 도중 특정 타이밍에 맞춘 수치 제어.  
* **예시:** 캐릭터가 발을 땅에 딛는 타이밍을 커브로 그려두고, Get Curve Value로 그 값을 읽어 발소리(SFX) 크기를 조절하거나 IK(Inverse Kinematics) 가중치를 조절합니다.

### **② 일반 블루프린트 (Curve Asset)에서의 역할**

별도로 생성한 **Curve Float 에셋**으로부터 값을 읽어올 때 사용합니다. (이때는 보통 커브 에셋 변수에서 드래그하여 Get Float Value라는 이름으로 쓰이기도 합니다.)

* **용도:** 시간에 따른 복잡한 수치 변화 구현.  
* **예시:** 수류탄의 대미지가 중심부에서 멀어질수록 어떻게 줄어드는지 곡선으로 그려두고, 거리에 따른 대미지 배율을 가져올 때 사용합니다.

## ---

**2\. 노드의 작동 원리 (Input & Output)**

* **Curve Name (입력):** 가져오고자 하는 커브의 이름을 정확히 입력합니다. (애니메이션 커브의 경우 에디터에서 설정한 이름과 일치해야 함)  
* **In Time (입력 \- 에셋 기준일 때):** 가로축(X축) 값입니다. 보통 '시간'이나 '진행률(0\~1)'을 넣습니다.  
* **Return Value (출력):** 해당 시점의 세로축(Y축) 값입니다. 곡선에 따라 계산된 결과값이 나옵니다.

## ---

**3\. 왜 사용하나요? (장점)**

### **"수학 대신 그래프로 그리세요"**

만약 어떤 값이 처음엔 천천히 변하다가 중간에 급격히 빨라지고, 마지막엔 튕기듯이 멈춰야 한다면 이를 코드로 짜기는 매우 복잡합니다. 하지만 **Get Curve Value**를 쓰면:

1. 커브 에디터에서 마우스로 \*\*점(Key)\*\*을 찍고 곡선을 예쁘게 그립니다.  
2. 블루프린트에서 노드 하나로 그 곡선 정보를 그대로 가져와 적용합니다.

## ---

**4\. 실무 활용 사례**

| 활용 분야 | 구체적인 예시 |
| :---- | :---- |
| **머티리얼 제어** | 애니메이션 커브 값을 읽어와 Set Scalar Parameter Value에 연결하여, 특정 동작 중 캐릭터의 몸이 빛나게 함. |
| **카메라 흔들림** | 커브에 불규칙한 파동을 그려두고, 카메라의 위치나 회전값에 더해 자연스러운 흔들림 구현. |
| **UI 연출** | 팝업창이 나타날 때 살짝 커졌다가 작아지는 탄성 효과(Elasticity)를 커브로 제어. |
| **성장 시스템** | 레벨이 오를수록 필요 경험치가 기하급수적으로 늘어나는 곡선을 읽어올 때. |

## ---

**💡 팁: Curve Name 오타 주의\!**

애니메이션 블루프린트에서 이 노드를 쓸 때 가장 흔한 실수는 커브 이름을 직접 타이핑하다가 생기는 **오타**입니다. 이름이 하나라도 틀리면 값은 무조건 **0**으로 출력됩니다. 가급적이면 커브 이름을 변수화하거나, 복사해서 붙여넣는 방식을 추천합니다.

**요약**

**Get Curve Value**는 "내가 그래프로 그려놓은 **수치의 흐름**을 그대로 가져와서 실제 게임 로직에 적용해 주는 **데이터 추출기**"입니다.
