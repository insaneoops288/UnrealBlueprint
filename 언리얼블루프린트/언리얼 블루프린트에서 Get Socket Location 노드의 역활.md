# **Get Socket Location 노드 상세 가이드**

언리얼 엔진 블루프린트에서 Get Socket Location 노드는 캐릭터의 손, 무기의 총구, 혹은 차량의 바퀴 등 특정 부위의 정확한 위치를 실시간으로 파악할 때 필수적인 노드입니다.

## **1\. 주요 역할**

이 노드는 지정된 메시 컴포넌트 내의 \*\*소켓(Socket)\*\*이나 \*\*본(Bone)\*\*의 위치를 **월드 공간(World Space)** 기준으로 반환합니다.

* **입력:** 타겟 메시 컴포넌트, 소켓 이름(Name)  
* **출력:** 벡터(Vector) 값 (X, Y, Z 좌표)

## **2\. 노드 파라미터 설명**

* **Target:** 소켓이 포함된 Scene Component(주로 스켈레탈 메시나 스태틱 메시)입니다.  
* **In Socket Name:** 위치를 찾고자 하는 소켓이나 본의 정확한 이름을 입력합니다. (대소문자 구분 없음)  
* **Return Value:** 해당 소켓의 현재 월드 좌표를 Vector 형태로 반환합니다.

## **3\. 대표적인 활용 사례**

### **A. 투사체 발사 (Projectile Spawning)**

총기 메시의 총구 부분에 Muzzle이라는 이름의 소켓을 만들고, Get Socket Location을 사용하여 총알이 생성될 정확한 위치를 결정합니다.

### **B. 이펙트 생성 (Particle Effects)**

캐릭터가 마법을 부릴 때 손바닥 위치에서 파티클이 나오게 하거나, 발걸음마다 먼지 효과를 줄 때 발 본(Bone)의 위치를 가져올 때 사용합니다.

### **C. 라인 트레이스 (Line Trace)**

검(Sword)의 시작점과 끝점에 소켓을 배치하고, 매 프레임마다 그 위치를 가져와 Line Trace By Channel 노드에 연결함으로써 무기 충돌 판정을 정밀하게 구현할 수 있습니다.

### **D. 오브젝트 부착 (Dynamic Attachment)**

런타임 중에 아이템을 캐릭터의 손에 붙이거나, 특정 위치로 이동시켜야 할 때 목표 지점의 좌표를 얻기 위해 사용합니다.

## **4\. 주의사항 및 팁**

1. **이름 확인:** 소켓 이름이 메시 에디터에서 설정한 이름과 정확히 일치해야 합니다. 오타가 있으면 보통 메시의 루트(Origin) 위치를 반환하게 됩니다.  
2. **본(Bone) 호환:** 이 노드는 '소켓'뿐만 아니라 스켈레톤의 '본' 이름도 인식합니다. 별도의 소켓을 만들지 않고 특정 손가락이나 관절의 위치를 바로 가져올 수도 있습니다.  
3. **Rotation과의 조합:** 위치뿐만 아니라 방향도 중요하다면 Get Socket Rotation이나 Get Socket Transform 노드를 함께 사용하여 회전값과 스케일값까지 한꺼번에 처리하는 것이 좋습니다.

## **5\. 요약**

Get Socket Location은 \*\*"지금 이 캐릭터(또는 물체)의 특정 부위가 월드 상 어디에 있는가?"\*\*라는 질문에 답해주는 노드입니다. 게임 내 물리 상호작용, 시각 효과, 시스템 로직의 정밀도를 높이는 데 핵심적인 역할을 합니다.
