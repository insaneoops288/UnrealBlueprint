# **Get Socket Transform 노드 상세 가이드**

언리얼 엔진에서 **Get Socket Transform** 노드는 캐릭터의 손에 무기를 쥐어주거나, 총구에서 총알이 발사되게 하는 등 특정 위치의 좌표 정보가 필요할 때 가장 많이 사용되는 노드입니다.

## **1\. 노드의 주요 역할**

이 노드는 특정 컴포넌트(주로 메쉬) 내에 정의된 **소켓(Socket)** 또는 \*\*본(Bone)\*\*의 **트랜스포름(위치, 회전, 스케일)** 값을 반환합니다.

* **소켓(Socket):** 개발자가 메쉬 에디터에서 직접 추가한 커스텀 부착 지점.  
* **본(Bone):** 모델링 단계에서 만들어진 스켈레톤의 뼈대 구조.

## **2\. 주요 입력 및 출력 항목 (Pins)**

### **입력 (Inputs)**

1. **Target:** 소켓이 포함된 **Scene Component**입니다. 주로 *Skeletal Mesh Component*나 *Static Mesh Component*가 연결됩니다.  
2. **In Socket Name:** 정보를 가져오고자 하는 소켓이나 본의 정확한 \*\*이름(Name)\*\*을 입력합니다. (대소문자 구분 없음)  
3. **Transform Space:** 어느 좌표계를 기준으로 값을 가져올지 결정합니다.  
   * **RTS\_World (World Space):** 게임 월드의 절대 좌표를 반환합니다. (가장 많이 사용됨)  
   * **RTS\_Actor (Actor Space):** 해당 액터의 루트를 기준으로 한 상대 좌표를 반환합니다.  
   * **RTS\_Component (Component Space):** 해당 메쉬 컴포넌트의 루트를 기준으로 한 상대 좌표를 반환합니다.

### **출력 (Outputs)**

* **Return Value:** 선택한 공간에서의 Transform 값(Location, Rotation, Scale)을 반환합니다.

## **3\. 주요 활용 사례**

### **① 무기 및 아이템 부착**

캐릭터의 손 소켓(Hand\_R\_Socket) 위치를 가져와서 무기 액터를 해당 위치로 이동시키거나 생성(Spawn)할 때 사용합니다.

### **② 발사체 및 이펙트 생성**

총기 메쉬의 총구 부분에 Muzzle 소켓을 만들고, 총알을 스폰하거나 총구 화염(VFX)을 재생할 위치와 방향을 정할 때 필수적입니다.

### **③ 카메라 시점 제어**

캐릭터의 머리 본(Head) 위치를 가져와 1인칭 카메라의 위치를 보정하거나, 특정 신체 부위를 추적하는 로직을 만들 때 사용합니다.

### **④ IK (Inverse Kinematics) 계산**

발바닥 소켓의 위치를 확인하여 지면과의 거리를 계산하고, 다리 관절의 굽힘을 조절하는 등 애니메이션 보정 로직의 기초 데이터로 활용됩니다.

## **4\. 주의사항 및 팁**

* **이름 오타 주의:** 소켓 이름이 틀리면 기본값(컴포넌트의 루트 위치)을 반환하므로, 메쉬 에디터에서 이름을 복사해서 붙여넣는 것이 좋습니다.  
* **성능 고려:** 매 프레임(Tick)마다 수십 개의 소켓 트랜스포름을 가져오는 것은 성능에 영향을 줄 수 있으므로 필요한 경우에만 호출하는 것이 효율적입니다.  
* **공간 설정 확인:** 가져온 값을 Spawn Actor from Class 노드 등에 연결할 때는 보통 **World Space**로 설정해야 액터가 월드 내 올바른 위치에 나타납니다.
