# **Is Any Montage Playing 노드 상세 가이드**

## **1\. 노드의 주요 역할**

Is Any Montage Playing 노드는 타겟으로 지정된 **애니메이션 인스턴스(Anim Instance)** 내에서 현재 실행 중인 몽타주가 하나라도 있는지를 체크하여 Boolean (True/False) 값으로 반환합니다.

* **True**: 현재 최소 하나 이상의 몽타주가 재생 중이거나 블렌드 아웃(Blend Out) 중일 때.  
* **False**: 어떠한 몽타주도 재생되고 있지 않을 때.

## **2\. 입력 및 출력**

* **Target (입력)**: Anim Instance 오브젝트 레퍼런스를 받습니다. 보통 Get Anim Instance 노드를 통해 연결합니다.  
* **Return Value (출력)**: 몽타주 재생 여부에 따라 True 또는 False를 반환합니다.

## **3\. 주요 활용 사례**

### **① 동작 중복 방지 (Input Guard)**

가장 흔한 사용 사례입니다. 캐릭터가 공격 몽타주를 재생 중일 때, 사용자가 공격 버튼을 연타하더라도 새로운 공격 애니메이션이 첫 번째 공격을 끊고 나가지 않도록 막을 때 사용합니다.

* **로직**: Branch 노드의 조건에 이 노드를 연결하여, False일 때만 새로운 공격 명령을 실행하도록 설정합니다.

### **② 상태 머신(State Machine) 전환 제어**

캐릭터의 상태(이동, 점프 등)를 변경하기 전, 특정 몽타주(예: 구르기, 피격)가 끝날 때까지 기다려야 하는 경우에 유용합니다.

### **③ UI 및 게임 로직 동기화**

몽타주가 재생되는 동안 플레이어의 입력을 무시하거나, 특정 UI 요소를 숨기는 등의 연출을 할 때 조건부로 활용됩니다.

## **4\. 주의사항 및 팁**

* **특정 몽타주 확인**: 만약 "어떤" 몽타주가 아니라 **"특정"** 몽타주가 재생 중인지 확인하고 싶다면 Montage Is Playing 노드를 사용해야 합니다.  
* **블렌드 시간**: 몽타주가 시각적으로는 거의 끝났더라도 설정된 'Blend Out' 시간 동안은 여전히 True를 반환할 수 있습니다.  
* **애니메이션 블루프린트**: 이 노드는 주로 캐릭터 블루프린트나 애니메이션 블루프린트 내부의 이벤트 그래프에서 자주 쓰입니다.

### **요약**

Is Any Montage Playing은 \*\*"캐릭터가 지금 무언가 특별한 동작(몽타주)을 수행하고 있는가?"\*\*를 묻는 가장 빠르고 간편한 방법입니다.
