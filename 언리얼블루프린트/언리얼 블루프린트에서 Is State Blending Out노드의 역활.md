언리얼 블루프린트에서 Is State Blending Out노드의 역활?

언리얼 엔진의 애니메이션 블루프린트(Anim Blueprint)에서 Is State Blending Out 노드는 현재 캐릭터가 머물고 있는 스테이트(State)가 다른 스테이트로 전환(Transition)되기 시작하여 서서히 빠져나가고 있는 중인지를 확인하는 역할을 합니다.

이 노드는 주로 스테이트 머신(State Machine) 내부의 로직이나 애니메이션 그래프에서 특정 상태가 종료되는 시점을 정교하게 제어할 때 사용됩니다.

1. 주요 역할 및 기능
전환 상태 감지: 현재 스테이트의 가중치(Weight)가 줄어들고 다음 스테이트의 가중치가 늘어나는 '블렌딩(Blending)' 과정이 시작되었을 때 True를 반환합니다.

종료 로직 실행: 애니메이션이 완전히 끝나기 직전, 즉 블렌딩이 시작되는 찰나에 특정 사운드를 재생하거나 변수 값을 변경하고 싶을 때 유용합니다.

2. 주요 사용 사례
연속 공격 시스템: 공격 애니메이션이 끝나고 다음 동작으로 넘어가는 블렌딩 시점에 "입력 가능" 플래그를 해제하거나 초기화할 때 사용합니다.

이펙트 및 사운드 제어: 특정 동작(예: 점프 착지) 스테이트가 끝나고 대기(Idle) 상태로 넘어갈 때, 발동 중이던 파티클을 끄거나 사운드를 페이드 아웃 시키는 용도로 활용합니다.

상태 전환 최적화: 스테이트가 완전히 빠져나가기 전(Blending Out 중)에 미리 다음 동작을 위한 데이터를 계산하거나 준비할 수 있습니다.

3. 유사한 노드와 비교
노드 이름	역할
Is State Blending Out	현재 스테이트가 **끝나가는 중(나가는 중)**인지 확인
Is State Blending In	현재 스테이트로 들어오는 중인지 확인
Get Relevant Anim Time Remaining	현재 애니메이션의 남은 시간을 확인 (전환 조건식에서 주로 사용)
4. 사용 시 주의사항
이 노드는 해당 스테이트 내부 또는 해당 스테이트를 참조할 수 있는 컨텍스트 내에서 호출되어야 정확한 값을 반환합니다. 보통 스테이트 머신의 전환 규칙(Transition Rule)보다는, 애니메이션 블루프린트의 Update Animation 이벤트나 스테이트 내부의 커스텀 블루프린트 로직에서 현재 상태를 모니터링할 때 자주 쓰입니다.

애니메이션이 자연스럽게 연결되도록 '블렌드 타임(Blend Time)'을 설정했을 때, 그 시간 동안 이 노드는 True를 유지하게 됩니다. 이를 통해 시각적으로 애니메이션이 섞이는 구간 동안 필요한 처리를 수행할 수 있습니다.
