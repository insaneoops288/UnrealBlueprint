언리얼 엔진의 애니메이션 블루프린트(AnimBP)에서 **Is State Blending Out** 노드는 현재 특정 상태(State)가 **다른 상태로 전환(Transition)되기 시작하여 점점 사라지고 있는 중인지**를 확인하는 질문 노드입니다.

단순히 "이 애니메이션이 재생 중인가?"를 묻는 것이 아니라, "이 애니메이션이 이제 **바톤을 터치하고 물러나는 단계인가?**"를 체크할 때 사용합니다.

## ---

**1\. 핵심 작동 원리**

상태 머신(State Machine)에서 A 상태에서 B 상태로 넘어갈 때, 보통 '블렌드 시간(Blend Time)'을 설정합니다. (예: 0.2초 동안 섞임)

* **Blending Out (True):** A 상태가 B 상태에게 자리를 내주기 위해 투명도(가중치)가 낮아지고 있는 그 0.2초 동안 이 값은 True가 됩니다.  
* **False:** 전환이 아예 시작되지 않았거나(완전히 재생 중), 이미 전환이 끝나서 완전히 사라진 상태라면 False를 반환합니다.

## ---

**2\. 왜 사용하나요? (주요 활용 사례)**

### **① 전환 타이밍에 맞춘 로직 실행**

애니메이션이 완전히 끝나기 직전, 즉 **다른 상태로 넘어가기 시작하는 찰나**에 무언가를 처리하고 싶을 때 유용합니다.

* *예: "공격 애니메이션이 끝나고 대기(Idle) 상태로 블렌딩 아웃되기 시작할 때, 캐릭터의 무적 판정을 꺼라."*

### **② 이펙트 및 사운드 정리**

특정 상태가 유지되는 동안만 나와야 하는 파티클이나 루프 사운드가 있을 때, 블렌딩 아웃이 시작되는 시점에 서서히 소리를 줄이거나 이펙트를 소멸시키는 용도로 사용합니다.

### **③ 특정 상태로의 재진입 방지**

이미 블렌딩 아웃이 진행 중인 상태(곧 사라질 상태)에서는 다시 그 상태의 특정 로직이 실행되지 않도록 가드(Guard) 역할을 하는 조건문으로 활용됩니다.

## ---

**3\. 노드 구성 방법**

이 노드는 보통 \*\*상태 머신 내부의 트랜지션 그래프(Transition Graph)\*\*가 아닌, **애니메이션 블루프린트의 이벤트 그래프나 업데이트 로직**에서 사용됩니다.

* **State Machine Name:** 확인하고 싶은 상태 머신의 이름을 선택합니다.  
* **State Name:** 감시하고 싶은 구체적인 상태(예: Attack\_01)의 이름을 선택합니다.  
* **Return Value:** 블렌딩 아웃 중이면 True, 아니면 False.

## ---

**4\. 유사 노드와 비교**

| 노드 이름 | 체크 시점 |
| :---- | :---- |
| **Is State Blending In** | 새로운 상태가 들어오기 시작할 때 (점점 나타나는 중) |
| **Is State Blending Out** | 현재 상태가 나가기 시작할 때 (점점 사라지는 중) |
| **Get Relevant Anim Time Remaining** | 애니메이션의 물리적인 남은 시간(초)을 체크할 때 |

## ---

**💡 실무 팁: "애니메이션 알림(Notify)과 조합"**

보통은 Anim Notify를 써서 특정 시점에 이벤트를 발생시키지만, **Is State Blending Out**은 '상태 머신의 흐름' 그 자체에 의존합니다. 따라서 애니메이션 파일을 일일이 열어서 노티파이를 찍기 번거롭거나, 어떤 애니메이션이 오든 **"이 상태를 벗어날 때"** 공통적으로 처리해야 할 전역 로직이 있다면 이 노드가 훨씬 효율적입니다.

**요약**

**Is State Blending Out**은 "지금 이 상태가 퇴장 준비 중인가?"를 알려주는 체크 노드입니다. 상태 전환 시 발생하는 부드러운 연결 구간을 감지하여 로직을 정밀하게 제어하고 싶을 때 꺼내 쓰시면 됩니다.

현재 작업 중인 성경 애니메이션 프로젝트에서 특정 동작(예: 기도하는 자세에서 일어나는 동작)이 끝나는 타이밍에 맞춰 이펙트를 제어하려고 하시나요? 이 노드를 활용하면 전환이 시작되는 부드러운 시점을 정확히 잡아낼 수 있습니다. 관련해서 더 구체적인 연결 방식이 궁금하신가요?
