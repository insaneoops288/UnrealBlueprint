# **언리얼 블루프린트: Make Vector 2D 노드 이해하기**

Make Vector 2D 노드는 개별적인 데이터 조각을 하나의 2D 좌표 체계로 묶어주는 아주 기초적이면서도 필수적인 노드입니다.

## **1\. 노드의 역할**

이 노드는 두 개의 입력 핀(**X**, **Y**)을 받아 하나의 **Vector 2D** 구조체 출력으로 내보냅니다.

* **X (Float)**: 가로축(Horizontal) 값  
* **Y (Float)**: 세로축(Vertical) 값  
* **Return Value (Vector 2D)**: 합쳐진 2D 벡터 데이터

## **2\. 주요 활용 사례**

### **① UI 및 위젯(UMG) 제어**

가장 많이 사용되는 분야입니다. 화면상의 위치나 크기를 조절할 때 필요합니다.

* **Set Position in Slot**: 위젯의 위치를 동적으로 변경할 때 X, Y 좌표를 생성하여 연결합니다.  
* **Set Render Scale**: 이미지나 버튼의 크기를 가로/세로 비율로 조절할 때 사용합니다.

### **② 입력 처리 (Input Handling)**

사용자의 마우스 이동이나 조이스틱 입력을 처리할 때 사용합니다.

* 마우스의 X 이동량과 Y 이동량을 합쳐서 화면상의 이동 벡터를 만들 때 사용합니다.

### **③ 머티리얼 및 UV 조작**

텍스트 좌표(UV)를 블루프린트에서 제어할 때 유용합니다.

* **Texture Offset**: 텍스트의 타일링이나 오프셋 값을 실시간으로 변경하기 위해 벡터 데이터를 생성합니다.

### **④ 2D 게임 로직**

2차원 평면 기반의 게임(플랫폼 게임 등)에서 캐릭터의 속도(Velocity)나 발사체의 방향을 정의할 때 사용합니다.

## **3\. 반대되는 노드: Break Vector 2D**

Make Vector 2D가 합치는 역할이라면, 반대로 2D 벡터 구조체에서 X와 Y 값을 다시 따로 추출하고 싶을 때는 **Break Vector 2D** 노드를 사용합니다.

## **4\. 팁**

* 핀 위에서 마우스 오른쪽 버튼을 클릭하고 \*\*'Split Struct Pin' (구조체 핀 분할)\*\*을 선택하면, 굳이 이 노드를 꺼내지 않고도 대상 노드에서 직접 X, Y 값을 입력할 수 있게 바뀝니다. (결과적으로 Make Vector 2D를 쓴 것과 같은 효과를 냅니다.)
