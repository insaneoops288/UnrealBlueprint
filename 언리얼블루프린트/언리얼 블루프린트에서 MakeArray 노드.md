언리얼 블루프린트 MakeArray 노드?

언리얼 엔진 블루프린트에서 Make Array 노드는 여러 개의 개별 항목을 모아 하나의 **배열(Array)**로 만들어주는 아주 유용한 노드입니다.

변수를 따로 만들지 않고도 즉석에서 배열을 생성해 다른 노드에 전달할 때 주로 사용하죠.

1. 주요 특징
가변적 입력 핀: 노드 왼쪽의 [+] (Add pin) 버튼을 눌러 원하는 만큼 입력 항목을 늘릴 수 있습니다. 반대로 필요 없는 핀은 우클릭해서 제거할 수 있습니다.

데이터 타입 자동 결정: 처음 연결하는 데이터의 타입에 따라 노드의 성격이 결정됩니다. 예를 들어, 정수(Integer)를 먼저 연결하면 해당 노드는 '정수 배열'을 만드는 노드가 됩니다.

복사본 생성: 이 노드는 실행될 때마다 설정된 값들을 바탕으로 새로운 배열을 생성하여 내보냅니다.

2. 사용 방법
그래프에서 우클릭 후 Make Array를 검색해 배치합니다.

Add pin 버튼을 눌러 배열에 넣고 싶은 개수만큼 칸을 만듭니다.

각 핀에 값이나 변수를 연결합니다.

오른쪽의 출력 핀을 배열을 필요로 하는 노드(예: For Each Loop, Set Array Elem 등)에 연결합니다.

3. 언제 사용하나요?
일회성 데이터 전달: 특정 함수에 여러 값을 한꺼번에 넘겨야 하는데, 굳이 변수로 저장해 둘 필요가 없을 때 유용합니다.

선택지 목록 만들기: 예를 들어, "랜덤하게 소환할 액터 클래스 목록"을 즉석에서 만들어 Get (Copy) 노드와 함께 사용할 수 있습니다.

순차적 로직 처리: For Each Loop와 연결하여 특정 아이템 세트나 타겟들을 차례대로 처리할 때 자주 쓰입니다.

💡 주의할 점
Make Array 노드는 호출될 때마다 배열을 새로 생성합니다. 만약 루프(Loop) 내부에서 이 노드를 반복적으로 사용하면 매번 메모리 할당이 일어나 성능에 영향을 줄 수 있으니, 데이터가 고정되어 있다면 변수로 만들어 사용하는 것이 더 효율적입니다.

혹시 특정 상황(예: 랜덤 아이템 스폰, UI 리스트 생성 등)에서 이 노드를 어떻게 활용하는지 구체적인 예시가 필요하신가요? 구체적인 상황을 알려주시면 바로 가이드를 드릴 수 있어요!
