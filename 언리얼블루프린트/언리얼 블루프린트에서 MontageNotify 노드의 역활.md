# **언리얼 블루프린트: Montage Notify의 역할과 활용**

언리얼 엔진에서 \*\*Montage Notify(몽타주 노티파이)\*\*는 애니메이션 몽타주가 재생되는 도중 특정 프레임(시점)에서 블루프린트 로직을 트리거하기 위해 사용되는 이벤트 시스템입니다.

## **1\. 핵심 역할: 애니메이션과 로직의 동기화**

가장 큰 역할은 **"시각적 동작(애니메이션)"과 "기능적 동작(코드/블루프린트)"을 일치시키는 것**입니다.

* **타이밍 제어:** 캐릭터가 칼을 휘두를 때, 실제로 데미지가 계산되어야 하는 정확한 시점을 알려줍니다.  
* **사운드 및 이펙트:** 발이 땅에 닿는 순간 먼지 파티클을 생성하거나 발소리 SFX를 재생합니다.  
* **상태 전환:** 애니메이션이 끝날 때까지 기다리지 않고, 특정 지점에서 콤보 입력을 허용하거나 이동 제한을 해제할 수 있습니다.

## **2\. 주요 노드: Play Montage with Notify**

일반적인 Play Montage 노드와 달리, Play Montage with Notify 노드는 몽타주 내에서 설정한 **Montage Notify**를 직접 수신할 수 있는 추가적인 실행 핀을 제공합니다.

### **노드의 출력 핀 구성**

1. **On Completed:** 몽타주가 끝까지 정상적으로 재생되었을 때 실행됩니다.  
2. **On Blended Out:** 몽타주가 다른 애니메이션으로 섞이며 나갈 때(블렌딩 아웃) 실행됩니다.  
3. **On Interrupted:** 재생 중 다른 몽타주에 의해 끊기거나 강제 종료되었을 때 실행됩니다.  
4. **On Notify Begin:** 몽타주 타임라인에서 **Montage Notify** 마커에 도달한 **순간** 실행됩니다.  
5. **On Notify End:** **Montage Notify Tick**과 같이 지속 시간이 있는 노티파이가 끝나는 시점에 실행됩니다.

## **3\. Montage Notify vs. 일반 Anim Notify**

언리얼에는 일반 Anim Notify와 Montage Notify가 있습니다. 이 둘의 차이점을 이해하는 것이 중요합니다.

| 구분 | 일반 Anim Notify (AN) | Montage Notify (MN) |
| :---- | :---- | :---- |
| **정의 위치** | 애니메이션 시퀀스/몽타주 모두 가능 | 주로 몽타주에서 사용 |
| **수신 장소** | \*\*애니메이션 블루프린트(AnimBP)\*\*의 Event Graph | 몽타주를 실행한 **액터 블루프린트** (Play Montage 노드) |
| **용도** | 범용적인 효과 (발소리, 파티클 등) | 해당 몽타주 재생 로직에 특화된 이벤트 (공격 판정 등) |

## **4\. 사용 방법 (Step-by-Step)**

1. **몽타주 편집:** 애니메이션 몽타주 에디터를 엽니다.  
2. **노티파이 트랙 추가:** Notifies 트랙에서 우클릭하여 Add Notify... \-\> Montage Notify를 선택합니다.  
3. **이름 설정:** 노티파이를 클릭하고 디테일 패널에서 Notify Name을 설정합니다 (예: DoDamage).  
4. **블루프린트 호출:** 캐릭터 블루프린트에서 Play Montage with Notify 노드를 배치하고 해당 몽타주를 할당합니다.  
5. **로직 연결:** On Notify Begin 핀에서 나온 실행 줄을 Branch나 Switch on Name 노드에 연결하여, 설정한 Notify Name이 맞는지 확인 후 원하는 로직(데미지 처리 등)을 작성합니다.

## **5\. 실무 활용 팁**

* **공격 판정:** On Notify Begin에서 Sphere Trace 등을 사용하여 공격 범위를 체크합니다.  
* **콤보 시스템:** 특정 노티파이 구간(Notify Window) 동안만 플레이어의 입력을 유효하게 처리하여 연계 공격을 구현합니다.  
* **발사체 생성:** 궁수가 화살을 놓는 정확한 프레임에 SpawnActorFromClass를 호출합니다.

결론적으로 **Montage Notify**는 애니메이션의 흐름을 게임의 데이터 및 로직과 연결해주는 매우 유연하고 직관적인 도구입니다.
