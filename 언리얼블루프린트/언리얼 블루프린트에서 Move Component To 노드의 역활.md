# **Move Component To 노드 가이드**

언리얼 엔진 블루프린트의 Move Component To 노드는 특정 **Scene Component**를 현재 위치에서 목표 위치(Location)와 회전(Rotation)으로 부드럽게 전환시키는 기능을 제공합니다.

## **1\. 주요 역할 (Role)**

이 노드의 핵심은 \*\*"시간에 따른 부드러운 이동과 회전"\*\*입니다. 단순히 위치값을 한 번에 바꾸는 Set Relative Location과 달리, 지정된 시간(Over Time) 동안 값을 보간하여 애니메이션 효과를 줍니다.

## **2\. 입력 핀 (Inputs) 설명**

| 입력 이름 | 설명 |
| :---- | :---- |
| **Component** | 이동시킬 대상 컴포넌트입니다. (예: Static Mesh, Camera, Scene 등) |
| **Target Relative Location** | 이동할 목표 **상대 좌표**입니다. (부모 컴포넌트 기준) |
| **Target Relative Rotation** | 회전할 목표 **상대 회전값**입니다. |
| **Ease In** | 체크 시, 이동 시작 부분에서 가속도가 붙으며 부드럽게 시작합니다. |
| **Ease Out** | 체크 시, 도착 지점에 가까워질수록 속도가 줄어들며 부드럽게 멈춥니다. |
| **Over Time** | 이동이 완료될 때까지 걸리는 시간(초 단위)입니다. |
| **Force Shortest Rotation Path** | 체크 시, 회전할 때 가장 짧은 경로(각도)를 선택하여 회전합니다. |

## **3\. 출력 핀 (Outputs) 설명**

| 출력 이름 | 설명 |
| :---- | :---- |
| **Completed** | 이동 및 회전이 완전히 끝났을 때 한 번 실행됩니다. 다음 로직을 연결하기 좋습니다. |

## **4\. 주요 특징 및 주의사항**

### **① 상대 좌표 기반 (Relative)**

이 노드는 **Relative(상대)** 좌표를 기준으로 작동합니다. 즉, 월드 좌표가 아니라 해당 컴포넌트의 부모(Parent)를 기준으로 한 좌표값으로 이동합니다. 부모가 없는 최상위 루트 컴포넌트라면 월드 좌표처럼 작동할 수 있습니다.

### **② 인터럽트(Interrupt) 가능**

노드가 실행 중일 때 다시 한번 호출되면, 현재 위치에서 새로운 목표치를 향해 다시 계산을 시작합니다.

### **③ 단순한 애니메이션에 최적**

타임라인(Timeline) 노드를 사용해 복잡한 커브를 그리거나 중간에 멈추는 세밀한 제어가 필요한 경우가 아니라면, 문(Door) 열기, UI 팝업 등장, 단순한 카메라 이동 등에 매우 빠르고 간편하게 사용할 수 있습니다.

## **5\. 실무 활용 예시**

1. **자동문 만들기:** 플레이어가 근처에 오면 Target Relative Location을 문이 열린 위치로 설정하고 Over Time을 0.5초 정도로 주어 부드럽게 엽니다.  
2. **카메라 전환:** 1인칭 뷰에서 조준(ADS) 시 카메라 위치를 총기 사이트로 살짝 옮길 때 유용합니다.  
3. **아이템 획득 연출:** 아이템을 먹었을 때 인벤토리 UI 방향이나 캐릭터 머리 위로 부드럽게 떠오르게 할 때 사용합니다.

## **요약**

Move Component To는 **코드가 간결**하면서도 **부드러운 움직임**을 만들어낼 수 있는 매우 효율적인 노드입니다. 복잡한 수학적 계산(Lerp 등) 없이도 자연스러운 연출이 가능하므로, 단순 이동 로직에서 가장 먼저 고려되는 노드입니다.
