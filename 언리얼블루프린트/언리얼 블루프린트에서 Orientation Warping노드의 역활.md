# **언리얼 엔진 Orientation Warping 노드 분석**

Orientation Warping은 **애니메이션 워핑(Animation Warping)** 기술 중 하나로, 캐릭터의 루트(Root) 방향과 실제 이동(Velocity) 방향 사이의 각도 차이를 보정하기 위해 하체 골격을 실시간으로 회전시키는 노드입니다.

## **1\. 주요 역할 (Core Functions)**

* **절차적 스트레이핑(Procedural Strafing) 구현**: 캐릭터가 앞을 보고 옆으로 걸을 때, 발의 방향이 진행 방향을 향하도록 골반과 다리 뼈대를 비틉니다.  
* **애니메이션 에셋 절약**: 45도, 90도 방향의 별도 애니메이션 없이 '앞으로 걷기' 에셋 하나만으로도 좌우 이동을 자연스럽게 표현할 수 있습니다.  
* **발 미끄러짐(Foot Sliding) 방지**: 이동 속도와 방향에 맞춰 뼈대를 직접 변형하므로, 캡슐 컴포넌트의 실제 이동과 애니메이션 간의 괴리를 줄여줍니다.

## **2\. 작동 원리**

1. **각도 계산**: 캐릭터가 바라보는 방향(Rotation)과 이동하는 방향(Velocity) 사이의 각도(Orientation Angle)를 구합니다.  
2. **골격 변형**: 계산된 각도만큼 하위 뼈대(주로 펠비스와 다리)를 회전시킵니다.  
3. **상체 고정**: 하체가 회전하더라도 상체(Spine)는 다시 반대 방향으로 보정하여 시선은 정면을 유지하게 만듭니다.

## **3\. 주요 파라미터 (Properties)**

| 파라미터명 | 역할 |
| :---- | :---- |
| **Orientation Angle** | 캐릭터 정면을 기준으로 이동 방향이 꺾인 각도(degree)입니다. 보통 Calculate Orientation Angle 함수를 통해 값을 전달받습니다. |
| **Spine Bones** | 하체가 돌아갈 때 상체를 반대로 회전시켜 고정할 척추 뼈들의 리스트입니다. |
| **IK Foot Root / Pelvis Bone** | 회전의 기준이 되는 골반 뼈와 발의 루트 뼈를 지정합니다. |
| **Rotation Linterp Speed** | 각도 변화에 따른 회전 속도를 조절하여 애니메이션이 너무 급격하게 변하지 않도록 합니다. |

## **4\. 사용 시 장점**

* **유연성**: 다양한 이동 속도와 각도에 즉각적으로 대응합니다.  
* **성능 최적화**: 수십 개의 애니메이션을 블렌딩하는 것보다 연산 효율이 높을 수 있습니다.  
* **시각적 품질**: 발이 땅을 딛는 방향이 진행 방향과 일치하여 훨씬 사실적으로 보입니다.

## **5\. 연관 노드**

오리엔테이션 워핑은 보통 단독으로 쓰이기보다 아래 노드들과 세트로 사용됩니다.

* **Stride Warping**: 보폭을 이동 속도에 맞춰 늘리거나 줄입니다.  
* **Slope Warping**: 경사면에서 발의 위치를 지면에 맞춥니다.  
* **Foot Placement**: 발이 지면에 정확히 닿도록 IK(Inverse Kinematics)를 적용합니다.

**요약:** Orientation Warping은 캐릭터가 "앞을 보면서 옆이나 뒤로 걸을 때" 애니메이션이 어색하지 않도록 **하체를 진행 방향으로 틀어주는 마법 같은 도구**입니다.
