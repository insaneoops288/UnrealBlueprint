# **Predict Ground Movement Stop Location 노드 가이드**

Predict Ground Movement Stop Location 노드는 캐릭터 무브먼트 컴포넌트(Character Movement Component)의 물리 법칙을 바탕으로, 캐릭터가 현재 이동 상태에서 **입력을 중단했을 때 최종적으로 멈추게 될 위치를 예측**하는 도구입니다.

## **1\. 주요 역할**

이 노드는 캐릭터가 가진 **관성, 마찰력, 제동 감속도**를 계산하여 미래의 정지 지점을 수학적으로 추정합니다. 단순히 현재 위치에 속도를 더하는 것이 아니라, 매 프레임 발생하는 감속 수치를 시뮬레이션하여 결과값을 내놓습니다.

## **2\. 입력 파라미터 (Inputs)**

| 파라미터 | 설명 |
| :---- | :---- |
| **Current Location** | 예측을 시작할 현재 위치입니다. |
| **Current Velocity** | 캐릭터의 현재 속도(Vector)입니다. 속도가 빠를수록 정지 거리가 길어집니다. |
| **Braking Deceleration** | 제동 감속도입니다. 값이 클수록 더 빨리 멈춥니다. (Character Movement의 Braking Deceleration Walking 값과 대응) |
| **Ground Friction** | 지면 마찰력입니다. 지면의 미끄러운 정도를 결정하며, 감속에 영향을 줍니다. |
| **Braking Friction Factor** | 제동 마찰 계수입니다. |

## **3\. 출력 결과 (Outputs)**

* **Return Value (Vector):** 캐릭터가 완전히 멈췄을 때의 예상 세계 좌표(World Location)를 반환합니다.

## **4\. 실전 활용 사례**

### **① AI의 경로 수정 및 충돌 방지**

AI 캐릭터가 매우 빠르게 달리고 있을 때, 목표 지점에 도달해서 멈추려고 하면 관성 때문에 목표를 지나칠 수 있습니다.

* **활용:** 이 노드로 정지 위치를 미리 계산하여, 목표 지점에 도달하기 전에 미리 감속을 시작하거나 경로를 수정하도록 설계할 수 있습니다.

### **② 애니메이션 전환 (Stop Transitions)**

캐릭터가 달리다가 멈출 때 '급정거' 애니메이션을 재생해야 하는 경우입니다.

* **활용:** 캐릭터가 곧 멈출 것이라는 것을 예측 지점을 통해 미리 감지하고, 실제 속도가 0이 되기 직전에 자연스러운 정지 애니메이션(Plant/Stop Animation)으로 블렌딩할 수 있습니다.

### **③ 시각적 가이드라인 (UI/VFX)**

레이싱 게임이나 정밀한 조작이 필요한 게임에서 플레이어에게 "지금 멈추면 여기서 멈춥니다"라는 가이드를 보여줄 때 사용합니다.

* **활용:** Draw Debug Sphere나 나이아가라 파티클을 예측된 위치에 배치하여 유저에게 제동 거리를 시각적으로 전달합니다.

## **5\. 주의사항**

* **캐릭터 무브먼트 전용:** 이 노드는 기본적으로 CharacterMovementComponent의 이동 로직(Walking 모드)에 최적화되어 있습니다. 일반적인 RigidBody 물리 시뮬레이션과는 계산 방식이 다를 수 있습니다.  
* **실시간 변화 미반영:** 예측 버튼을 누른 순간의 데이터를 기준으로 계산하므로, 예측 직후에 마찰력이 변하거나 경사로를 만나는 등의 외부 요인은 결과값에 반영되지 않습니다.
