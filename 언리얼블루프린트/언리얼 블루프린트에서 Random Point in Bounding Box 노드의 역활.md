# **Random Point in Bounding Box 노드 분석**

언리얼 엔진(Unreal Engine)의 블루프린트에서 Random Point in Bounding Box 노드는 정의된 3차원 박스 영역 내에서 \*\*무작위 위치(Vector)\*\*를 반환하는 함수입니다.

## **1\. 노드의 주요 입력항목 (Inputs)**

이 노드는 두 가지 핵심 데이터를 입력받아 범위를 계산합니다.

* **Origin (Vector):** 박스의 **중심점** 좌표입니다. 박스가 월드 어디에 위치할지를 결정합니다.  
* **Box Extent (Vector):** 박스의 **반지름(Half-size)** 값입니다. 중심점으로부터 각 축(X, Y, Z) 방향으로 얼만큼 떨어진 지점까지 범위를 잡을지 설정합니다.  
  * *주의:* 전체 크기가 200인 박스를 만들고 싶다면, Extent 값은 100을 입력해야 합니다.

## **2\. 노드의 출력항목 (Outputs)**

* **Return Value (Vector):** 계산된 범위 내의 임의의 **3D 좌표값**입니다.

## **3\. 작동 원리 (내부 로직)**

이 노드는 수학적으로 다음과 같은 계산을 수행하여 결과를 냅니다.

* **결과 X** \= Origin.X ± (0에서 BoxExtent.X 사이의 랜덤값)  
* **결과 Y** \= Origin.Y ± (0에서 BoxExtent.Y 사이의 랜덤값)  
* **결과 Z** \= Origin.Z ± (0에서 BoxExtent.Z 사이의 랜덤값)

즉, 지정된 중심점을 기준으로 설정한 범위(Extent) 내에서 가상의 박스를 그리고, 그 안의 점 하나를 찍어주는 것입니다.

## **4\. 주요 활용 사례**

1. **아이템/몬스터 스폰 (Spawning):**  
   * 특정 구역(예: 숲, 방 안) 내에서 적이나 아이템을 무작위 위치에 생성하고 싶을 때 사용합니다.  
   * 주로 Get Actor Bounds 노드와 결합하여 특정 액터의 부피 내에서 위치를 찾기도 합니다.  
2. **앰비언트 사운드/이펙트:**  
   * 나비가 날아다니는 이펙트나, 숲속의 새 소리가 특정 영역 안에서 들리게 할 때 무작위 위치를 지정하기 위해 사용합니다.  
3. **순찰 지점 설정 (AI Patrol):**  
   * AI 캐릭터가 특정 구역 내에서 무작위로 돌아다니게 하고 싶을 때, 다음 이동 목표 지점을 이 노드로 결정합니다.

## **5\. 팁: 박스 콜리전과 함께 사용하기**

가장 흔한 패턴은 **Box Collision** 컴포넌트를 가진 액터에서 해당 컴포넌트의 위치와 크기를 가져와 연결하는 방식입니다.

* Get Component Location → **Origin**에 연결  
* Get Scaled Box Extent → **Box Extent**에 연결

이렇게 하면 에디터 뷰포트에서 박스 컴포넌트의 크기를 조절하는 대로 랜덤 포인트 생성 범위가 실시간으로 반영되어 작업이 매우 편리해집니다.
