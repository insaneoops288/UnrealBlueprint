# **언리얼 엔진 Rotate Root Bone 노드 상세 가이드**

Rotate Root Bone 노드는 애니메이션 블루프린트의 애님 그래프(AnimGraph)에서 캐릭터의 가장 최상위 본인 \*\*루트 본(Root Bone)\*\*의 회전값을 동적으로 수정하는 역할을 합니다.

## **1\. 핵심 역할**

주로 캐릭터가 바라보는 방향과 실제 캡슐 컴포넌트(Capsule Component)의 방향 사이의 차이를 메우거나, 애니메이션 도중 루트 본의 방향을 강제로 조정해야 할 때 사용됩니다.

* **절차적 회전 제어:** 특정 애니메이션 파일 없이도 코드를 통해 캐릭터의 몸체 방향을 회전시킬 수 있습니다.  
* **루트 모션(Root Motion) 보정:** 루트 모션이 포함된 애니메이션에서 발생할 수 있는 회전 오차를 보정하거나, 회전 각도를 추가할 때 유용합니다.  
* **방향 동기화:** 캐릭터가 이동하는 방향(Velocity)이나 카메라가 바라보는 방향에 맞춰 하체를 고정한 채 상체나 전체적인 방향을 부드럽게 돌릴 때 사용됩니다.

## **2\. 주요 핀(Pins) 설명**

| 핀 이름 | 설명 |
| :---- | :---- |
| **Component Pose** | 입력받는 기본 애니메이션 포즈입니다. |
| **Pitch/Yaw/Roll** | 루트 본에 적용할 회전값입니다. 보통 변수를 연결하여 동적으로 제어합니다. |
| **Mesh Component Rotation** | 현재 스켈레탈 메시 컴포넌트의 월드 회전값입니다. 루프백(Loopback) 방지를 위해 참조됩니다. |
| **Root Bone Rotation** | 최종적으로 루트 본에 적용될 계산된 회전값입니다. |

## **3\. 주요 활용 사례**

### **① 제자리 회전 (Turn In Place)**

캐릭터가 제자리에서 회전할 때, 캡슐은 회전하지만 메시(애니메이션)는 즉각 따라가지 않게 설정하는 경우가 많습니다. 이때 Rotate Root Bone을 사용해 메시가 캡슐의 회전 속도를 부드럽게 따라오도록 오프셋을 줄 수 있습니다.

### **② 루트 모션의 방향 수정**

루트 모션 애니메이션이 직선으로만 되어 있을 때, 게임 로직에서 계산된 굴곡에 따라 루트 본을 회전시켜 캐릭터가 곡선 경로로 이동하도록 만들 수 있습니다.

### **③ 경사면 대응**

캐릭터가 경사로를 오를 때, 발이 지면에 닿는 각도에 맞춰 루트 본을 회전시켜 캐릭터의 서 있는 자세를 더 자연스럽게 만들 때 사용되기도 합니다 (다만, 이 경우는 보통 IK와 병행합니다).

## **4\. 특징 및 주의사항**

1. **루트 본 한정:** 이름 그대로 '루트 본'에만 영향을 미칩니다. 골반(Pelvis)이나 다른 본을 회전시키려면 Transform (Modify) Bone 노드를 사용해야 합니다.  
2. **좌표계 확인:** 회전이 로컬 공간(Local Space)에서 일어나는지, 컴포넌트 공간(Component Space)에서 일어나는지 노드 디테일 패널 설정을 확인해야 합니다.  
3. **캡슐과의 관계:** 루트 본을 회전시킨다고 해서 캐릭터의 물리 충돌체인 '캡슐 컴포넌트'가 함께 회전하는 것은 아닙니다. 비주얼적인 메시의 방향만 바뀌는 것임을 인지해야 합니다.

## **5\. 요약**

Rotate Root Bone 노드는 **"애니메이션 데이터에 정의된 회전값 외에, 게임 실행 중에 실시간으로 캐릭터의 뿌리(Root) 방향을 틀어주고 싶을 때"** 사용하는 도구입니다. 주로 부드러운 방향 전환이나 루트 모션 제어 시스템을 구축할 때 핵심적인 역할을 수행합니다.
