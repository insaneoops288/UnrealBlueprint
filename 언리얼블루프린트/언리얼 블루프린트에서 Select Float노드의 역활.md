언리얼 블루프린트에서 Select Float노드의 역활?

언리얼 엔진 블루프린트에서 Select Float 노드는 특정 조건(Boolean)에 따라 두 개의 실수(Float) 값 중 하나를 선택하여 출력하는 역할을 합니다.

쉽게 말해, 프로그래밍의 삼항 연산자(Ternary Operator) 또는 **"만약 ~라면 A를, 아니면 B를 선택해"**라는 논리를 데이터 흐름 안에서 구현하는 노드입니다.

1. 노드의 구조
이 노드는 크게 세 가지 입력 핀과 한 개의 출력 핀으로 구성됩니다.

A (Float): Pick A가 True일 때 출력될 값입니다.

B (Float): Pick A가 False일 때 출력될 값입니다.

Pick A (Boolean): 어떤 값을 선택할지 결정하는 스위치 역할입니다.

Return Value (Float): 선택된 최종 값이 이 핀을 통해 나갑니다.

2. 왜 사용하는가? (Branch 노드와의 차이점)
보통 조건에 따라 값을 바꿀 때 Branch(분기) 노드를 떠올리기 쉽지만, Select Float 노드는 다음과 같은 장점이 있습니다.

특징	Branch 노드	Select Float 노드
흐름 제어	실행 선(하얀 선)을 두 갈래로 나눔	실행 선은 그대로 두고 데이터만 선택
복잡도	변수 설정(Set) 노드를 중복해서 써야 함	노드 하나로 깔끔하게 처리 가능
용도	로직 전체의 흐름을 바꿀 때	특정 수치(데미지, 속도 등)만 바꿀 때
3. 실무 활용 예시
난이도에 따른 데미지 조절

IsHardMode라는 변수가 True면 데미지를 100.0, False면 50.0으로 설정하여 공격력 계산에 연결합니다.

질주 기능 (Walk/Run)

IsSprinting 변수에 따라 캐릭터의 Max Walk Speed를 600.0 또는 300.0으로 즉시 전환할 때 사용합니다.

UI 게이지 색상/수치 변경

체력이 일정 수치 이하인지 체크하여, 맞다면 낮은 수치를, 아니면 정상 수치를 출력하게 만듭니다.

4. 팁: 확장성
언리얼에는 Select Float뿐만 아니라 Select Int, Select Vector, Select Color 등 다양한 타입의 Select 노드가 있습니다. 또한, 단순히 Boolean(True/False)뿐만 아니라 **Integer나 Enum(열거형)**을 Index 핀에 연결하면 3개 이상의 값 중 하나를 선택할 수도 있는 강력한 Select 노드로 변신합니다.

요약하자면: "조건에 따라 하얀 실행 선을 쪼개지 않고, 녹색 데이터 선 하나에서 값만 쏙 골라 쓰고 싶을 때" 사용하는 아주 유용한 노드입니다.
