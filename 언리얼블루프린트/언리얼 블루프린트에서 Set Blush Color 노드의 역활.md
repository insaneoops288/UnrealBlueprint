# **Unreal Engine Blueprint: Set Brush Color 노드 가이드**

언리얼 엔진의 블루프린트에서 **Set Brush Color** (또는 UI 관련 노드)는 주로 **UMG(User Widget)** 내의 시각적 요소들의 색상과 투명도를 동적으로 제어하는 역할을 합니다.

## **1\. 노드의 주요 역할**

이 노드는 위젯 컴포넌트(예: Image, Border, Button)의 **Brush** 속성에 지정된 색상(Tint)을 변경합니다.

* **동적 시각 피드백**: 게임 플레이 중 캐릭터의 체력이 낮아지면 화면 테두리를 빨갛게 바꾸거나, 버튼에 마우스를 올렸을 때 색상이 변하는 효과를 줄 때 사용합니다.  
* **슬레이트 브러시(Slate Brush) 수정**: UI의 배경 이미지나 텍스트 박스의 배경색을 프로그래밍 방식으로 업데이트합니다.

## **2\. 주요 입력 파라미터**

* **Target (타겟)**: 색상을 변경하고자 하는 위젯 객체입니다 (주로 Image 또는 Border 위젯).  
* **In Color (입력 색상)**: 적용하려는 새로운 색상 값입니다. **Linear Color** 구조체를 사용하며, R(빨강), G(초록), B(파랑), A(알파/투명도) 값을 가집니다.

## **3\. 활용 예시**

### **A. 체력 바 색상 변경**

플레이어의 HP 수치에 따라 이미지의 색상을 녹색에서 빨간색으로 부드럽게 변경할 때 Lerp (Linear Color) 노드와 함께 Set Brush Color를 사용합니다.

### **B. 아이템 등급 표시**

인벤토리 시스템에서 아이템의 등급(일반, 희귀, 전설)에 따라 슬롯의 배경 테두리 색상을 다르게 설정할 때 유용합니다.

## **4\. 주의사항 (Blush vs Brush)**

만약 사용 중인 프로젝트가 특정 **캐릭터 커스터마이징 플러그인**이나 **메타휴먼** 관련 에셋이라면, 볼에 홍조를 띄게 하는 Set Blush Color라는 이름의 사용자 정의 변수나 함수가 존재할 수 있습니다.

* 이 경우 해당 노드는 캐릭터 머티리얼의 **Parameter(파라미터)** 값을 변경하여 피부의 특정 부위(뺨) 색상을 붉게 만드는 역할을 수행합니다.  
* 일반적인 엔진 표준 노드는 **Set Brush Color**가 맞으므로, 노드 검색 시 철자를 확인해 보시기 바랍니다.

## **5\. 블루프린트 구성 팁**

1. **변수 생성**: Linear Color 타입의 변수를 만들어 색상을 미리 정의해 두면 관리가 편합니다.  
2. **이벤트 연동**: On Mouse Enter나 On Clicked 같은 위젯 이벤트에 연결하여 즉각적인 반응형 UI를 만들 수 있습니다.
