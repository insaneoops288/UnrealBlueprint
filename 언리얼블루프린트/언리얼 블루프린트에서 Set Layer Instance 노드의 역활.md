# **언리얼 엔진 애니메이션 레이어: Set Layer Instance 역할 가이드**

언리얼 엔진 4.24 버전부터 도입된 **애니메이션 블루프린트 레이어(Animation Blueprint Layers)** 시스템은 복잡한 캐릭터 애니메이션을 체계적으로 관리하기 위한 강력한 도구입니다. 이 시스템의 핵심 노드인 Set Layer Instance의 역할을 상세히 알아봅니다.

## **1\. 핵심 정의**

Set Layer Instance는 **메인 애니메이션 블루프린트(Main AnimBP)** 내에 정의된 '애니메이션 레이어' 슬롯에, 특정 로직이 구현된 \*\*'레이어 인스턴스(Layer Instance)'\*\*를 실행 중에(Runtime) 동적으로 연결하거나 교체하는 역할을 합니다.

## **2\. 왜 사용하는가? (주요 이점)**

### **① 애니메이션 로직의 모듈화**

과거에는 모든 무기(권총, 소총, 칼 등)에 대한 애니메이션 로직을 하나의 메인 AnimBP 안에 거대한 State Machine이나 Blend by Enum으로 때려 넣어야 했습니다. 이는 그래프를 복잡하게 만들고 관리를 어렵게 합니다.

* **해결:** 무기별로 별도의 '레이어 블루프린트'를 만들고, 필요할 때만 Set Layer Instance로 갈아 끼웁니다.

### **② 협업의 용이성**

메인 캐릭터의 이동 로직을 담당하는 작업자와 특정 아이템(예: 제트팩, 탈것)의 애니메이션을 담당하는 작업자가 **동일한 파일을 건드리지 않고** 각자 작업할 수 있습니다.

### **③ 메모리 및 성능 최적화**

현재 캐릭터가 사용하지 않는 무기나 상태의 애니메이션 로직은 아예 로드하거나 계산하지 않으므로 성능 이득을 볼 수 있습니다.

## **3\. 작동 메커니즘**

이 노드를 사용하기 위해서는 보통 다음 3단계 구조를 가집니다.

### **단계 1: 애니메이션 레이어 인터페이스 (Animation Layer Interface)**

* "어떤 자리가 필요한가?"를 정의합니다.  
* 예: WeaponLayer(InputPose) 같은 함수 이름을 정의합니다.

### **단계 2: 메인 애니메이션 블루프린트 (Main AnimBP)**

* 인터페이스를 추가하고, 애님 그래프에서 **Linked Anim Layer** 노드를 배치하여 "여기에 외부 로직이 들어올 것"이라고 자리를 잡아둡니다.

### **단계 3: 레이어 전용 블루프린트 (Layer AnimBP)**

* 인터페이스를 구현(Implement)하여 실제 애니메이션(예: 소총 조준 포즈)을 작성합니다.

### **단계 4: 블루프린트 호출 (The Node)**

* 캐릭터 블루프린트에서 캐릭터가 소총을 집었을 때, Set Layer Instance 노드를 호출하여 **메인 AnimBP**의 특정 인터페이스 자리에 **소총용 레이어 블루프린트**를 할당합니다.

## **4\. 노드 주요 핀 설명**

| 핀 이름 | 역할 |
| :---- | :---- |
| **Target** | 해당 레이어를 설정할 대상 애님 인스턴스 (보통 Get Anim Instance 결과값) |
| **In Class** | 주입할 '애니메이션 레이어 블루프린트' 클래스 |
| **Layer Interface** | 어떤 인터페이스를 통해 연결할지 선택 |

## **5\. 실전 활용 사례**

* **장착 아이템 시스템:** 캐릭터가 가방을 메면 가방 전용 본(Bone) 조작 로직이 담긴 레이어를 연결.  
* **부상 시스템:** 캐릭터의 체력이 특정 수치 이하로 떨어지면 '절뚝거리는 걷기' 로직이 담긴 레이어로 교체.  
* **다양한 스타일:** 동일한 캐릭터가 '공격적 스타일' 또는 '방어적 스타일'의 포즈 세트를 동적으로 전환할 때.

## **6\. 주의사항**

* **인터페이스 일치:** 주입하려는 클래스가 메인 AnimBP에서 정의한 인터페이스를 정확히 상속받고 있어야 합니다.  
* **초기화 시점:** BeginPlay에서 바로 호출할 경우, 애님 인스턴스가 아직 유효하지 않을 수 있으므로 유효성 검사(Is Valid)가 필요할 수 있습니다.

**요약하자면:** Set Layer Instance는 마치 컴퓨터 본체(Main AnimBP)의 슬롯에 필요한 기능 카드(Layer AnimBP)를 상황에 맞춰 꽂았다 뺐다 하는 **스위치 역할**을 합니다.
