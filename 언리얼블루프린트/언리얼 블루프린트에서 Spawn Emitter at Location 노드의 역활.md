# **Spawn Emitter at Location 노드 상세 설명**

언리얼 엔진 블루프린트에서 **Spawn Emitter at Location** 노드는 월드 내의 특정 좌표에 파티클 시스템(Cascade 또는 Niagara)을 생성하는 데 사용됩니다.

## **1\. 주요 역할**

이 노드의 핵심은 \*\*"일회성 또는 독립적인 효과 생성"\*\*입니다. 특정 캐릭터나 오브젝트에 종속되지 않고, 지정된 위치에 효과를 뿌린 뒤 작업이 끝나면 자동으로 소멸하는 방식을 취합니다.

## **2\. 주요 입력 파라미터 (Inputs)**

| 파라미터 | 설명 |
| :---- | :---- |
| **Emitter Template** | 생성할 파티클 시스템 에셋을 선택합니다. (Cascade의 경우 Particle System, Niagara의 경우 Niagara System은 별도의 'Spawn System at Location' 사용 권장) |
| **Location** | 파티클이 생성될 월드 기준의 3D 좌표(Vector)입니다. |
| **Rotation** | 파티클이 방출될 방향(Rotator)을 설정합니다. |
| **Scale** | 파티클의 크기를 조절합니다. (기본값 1.0, 1.0, 1.0) |
| **Auto Destroy** | 효과가 종료되면 자동으로 컴포넌트를 메모리에서 제거할지 여부입니다. (기본값: True) |
| **Auto Activate** | 생성과 동시에 효과를 시작할지 설정합니다. (기본값: True) |

## **3\. 반환값 (Return Value)**

이 노드는 생성된 **Particle System Component**의 참조를 반환합니다. 이를 변수로 저장하면, 나중에 블루프린트에서 해당 파티클의 색상을 변경하거나, 강제로 중지시키는 등의 추가 제어가 가능합니다.

## **4\. 활용 사례**

* **폭발 효과:** 수류탄이나 미사일이 지면에 충격했을 때 그 위치에 폭발 효과 생성.  
* **피격 효과:** 캐릭터가 총에 맞았을 때 피격된 위치에 혈흔이나 스파크 생성.  
* **발자국 먼지:** 캐릭터가 뛸 때 발 위치에 먼지 파티클 생성.  
* **마법 캐스팅:** 마법진이 나타나야 하는 특정 위치에 시각 효과 생성.

## **5\. 주의사항 및 팁**

1. **성능 최적화:** Auto Destroy가 체크되어 있지 않으면 파티클이 눈에 보이지 않아도 메모리에 계속 남아 성능 저하를 유발할 수 있습니다.  
2. **Niagara 사용 시:** 언리얼 엔진 4.26 이후부터는 Cascade보다 Niagara가 권장됩니다. Niagara 효과를 쓸 때는 Spawn System at Location 노드를 사용하는 것이 더 적합합니다.  
3. **부착 여부:** 만약 파티클이 캐릭터의 손이나 총구에 따라다녀야 한다면 이 노드 대신 **Spawn Emitter Attached** 노드를 사용해야 합니다.

## **6\. 요약**

**Spawn Emitter at Location**은 "어디(Location)에 어떤 효과(Emitter)를 잠깐 보여주고 사라지게 할 것인가?"를 결정하는 가장 기본적인 노드입니다.
