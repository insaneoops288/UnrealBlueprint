# **언리얼 엔진 Stride Warping 노드 상세 분석**

**Stride Warping**은 캐릭터의 이동 속도(Actual Velocity)에 맞춰 다리 애니메이션의 보폭을 실시간으로 늘리거나 줄여주는 애니메이션 노드입니다.

## **1\. 핵심 역할: 발 미끄러짐(Foot Sliding) 방지**

전통적인 방식에서는 캐릭터의 이동 속도가 빨라지면 애니메이션 재생 속도(Play Rate)를 높였습니다. 하지만 이 경우 캐릭터가 '경보'를 하는 것처럼 부자연스럽게 보일 수 있습니다.

Stride Warping은 재생 속도를 조절하는 대신, **다리의 관절 위치(IK)를 조정하여 보폭 자체를 수정**함으로써 실제 이동 거리와 애니메이션상의 발 위치를 정확히 일치시킵니다.

## **2\. 주요 기능 및 동작 원리**

### **A. 속도 비율 계산**

노드는 내부적으로 다음 두 수치를 비교합니다.

* **애니메이션 속도 (Gait Speed):** 애니메이션 데이터 자체에 기록된 이동 속도.  
* **실제 이동 속도 (World Speed):** 캐릭터 무브먼트 컴포넌트 등을 통해 들어오는 현재 캐릭터의 실제 속도.

이 두 값의 비율을 바탕으로 다리를 얼마나 더 뻗어야 할지(Extension)를 결정합니다.

### **B. 보폭 수정 (Stride Adjustment)**

* **속도가 빠를 때:** 다리를 더 멀리 뻗도록 골반과 발 위치를 조정합니다.  
* **속도가 느릴 때:** 다리를 좁게 뻗도록 조정합니다.

### **C. IK(Inverse Kinematics) 활용**

Stride Warping은 보통 **IK 리깅**과 함께 작동합니다. 골반(Hip)의 높이를 유지하면서 발(Foot)의 위치를 전후로 밀어주기 때문에, 캐릭터가 단순히 커지는 것이 아니라 지면을 박차고 나가는 느낌을 유지합니다.

## **3\. 주요 파라미터 설명**

| 파라미터 | 설명 |
| :---- | :---- |
| **Stride Direction** | 보폭이 조정될 방향을 지정합니다. (보통 캐릭터의 정면 방향) |
| **Stride Value** | 워핑 강도를 결정합니다. 1.0은 실제 속도에 100% 맞춤을 의미합니다. |
| **Pivot Spine / Pelvis** | 보폭이 변할 때 골반이나 척추가 어느 정도의 보상 움직임을 가질지 설정합니다. |
| **Gait Speed** | 현재 애니메이션이 상정하는 기준 속도입니다. |
| **IK Foot Root / Bones** | 워핑을 적용할 발 뼈와 IK 타겟 뼈를 지정합니다. |

## **4\. 사용 시 이점**

1. **비주얼 퀄리티 향상:** 발이 지면에 붙어있는 것처럼 보이게 하여 몰입감을 높입니다.  
2. **애니메이션 자산 절약:** 하나의 걷기/달리기 애니메이션으로 다양한 이동 속도 범위를 자연스럽게 커버할 수 있습니다.  
3. **동적 대응:** 가속/감속 구간이나 경사지에서도 발 미끄러짐 없이 부드러운 전환이 가능합니다.

## **5\. 요약**

Stride Warping은 \*\*"캐릭터가 초당 이동하는 거리"\*\*와 \*\*"애니메이션 한 걸음의 거리"\*\*를 강제로 일치시켜주는 도구입니다. 이는 특히 루트 모션(Root Motion)을 사용하지 않는 인플레이스(In-place) 애니메이션 시스템에서 고품질의 캐릭터 움직임을 구현할 때 필수적인 노드입니다.
