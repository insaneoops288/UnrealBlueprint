# **Switch On Int 노드 이해하기**

**Switch On Int** 노드는 복잡한 조건문(Branch)을 간결하게 만들어주는 흐름 제어(Flow Control) 노드입니다.

## **1\. 주요 역할**

* **다중 분기 처리**: 하나의 입력값(정수)을 확인하여, 그 값에 해당하는 출력 핀으로 실행 흐름을 보냅니다.  
* **코드 가독성 향상**: 여러 개의 Branch 노드를 중첩해서 사용하는 대신, 하나의 노드로 깔끔하게 관리할 수 있습니다.

## **2\. 노드의 구성 요소**

* **Selection (입력 핀)**: 비교의 기준이 되는 정수 변수나 값을 연결합니다.  
* **Execution Pins (출력 핀)**: 각 숫자(0, 1, 2...)에 대응하는 실행 경로입니다.  
* **Default (기본 핀)**: 입력값이 설정된 어떤 케이스에도 해당하지 않을 때 실행되는 '예외 처리' 경로입니다.

## **3\. 주요 설정 및 조작법**

* **핀 추가 (Add Pin)**: 노드 하단의 'Add Pin' 버튼을 클릭하여 실행 경로를 늘릴 수 있습니다. (예: 0, 1, 2, 3...)  
* **시작 인덱스 설정**: 디테일 패널에서 Start Index를 변경하여 시작 숫자를 조절할 수 있습니다 (기본값은 0).  
* **핀 제거**: 필요 없는 출력 핀 위에서 우클릭하여 제거할 수 있습니다.

## **4\. 실전 활용 사례**

* **캐릭터 상태 제어**:  
  * 0: 대기(Idle)  
  * 1: 걷기(Walk)  
  * 2: 달리기(Run)  
  * 3: 점프(Jump)  
* **게임 난이도 설정**: 난이도를 0(쉬움), 1(보통), 2(어려움)로 구분하여 적의 체력이나 공격력을 다르게 설정할 때 유용합니다.  
* **아이템 등급**: 아이템의 등급 코드를 정수로 관리하여 각 등급에 맞는 이펙트나 사운드를 재생할 때 사용합니다.

## **5\. Branch vs Switch**

| 특징 | Branch (분기) | Switch On Int (정수 스위치) |
| :---- | :---- | :---- |
| **비교 방식** | 참(True) 또는 거짓(False) | 특정 숫자값 일치 여부 |
| **적합한 상황** | 조건이 하나인 단순한 판단 | 조건이 3개 이상인 다중 선택 상황 |

**Tip**: 만약 정수 대신 '열거형(Enum)'을 사용한다면 Switch on (Enum Name) 노드를 사용하는 것이 훨씬 직관적입니다. 하지만 동적인 숫자 계산 결과에 따라 경로를 나눌 때는 Switch On Int가 필수적입니다.
