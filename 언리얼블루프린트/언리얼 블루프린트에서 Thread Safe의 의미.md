# **언리얼 블루프린트에서의 Thread Safe(스레드 안전) 의미**

언리얼 엔진에서 **Thread Safe**란, 여러 개의 스레드가 동시에 같은 데이터에 접근하거나 함수를 실행해도 데이터가 오염되지 않고 프로그램이 충돌(Crash)하지 않는 상태를 의미합니다. 특히 블루프린트에서는 \*\*애니메이션 블루프린트(Animation Blueprint)\*\*의 최적화 과정에서 이 용어를 가장 자주 접하게 됩니다.

## **1\. 핵심 배경: 게임 스레드 vs 워커 스레드**

일반적으로 블루프린트의 대부분의 로직(이벤트 그래프 등)은 \*\*게임 스레드(Game Thread)\*\*라고 불리는 메인 스레드에서 실행됩니다. 하지만 엔진 성능을 높이기 위해 언리얼은 애니메이션 계산과 같은 무거운 작업을 별도의 \*\*워커 스레드(Worker Thread)\*\*로 분산시키는데, 이를 '멀티 스레드 애니메이션 업데이트'라고 합니다.

* **문제 발생:** 워커 스레드에서 실행되는 로직이 메인 게임 스레드에 있는 변수(예: 캐릭터의 HP, 현재 속도 등)를 직접 수정하거나 읽으려고 하면, 두 스레드가 충돌하여 게임이 꺼지거나 값이 이상해질 수 있습니다.  
* **해결책:** 워커 스레드에서도 안심하고 실행할 수 있도록 설계된 함수나 변수 접근 방식을 **Thread Safe**하다고 표현합니다.

## **2\. 블루프린트에서 Thread Safe의 주요 특징**

### **① 애니메이션 블루프린트에서의 활용**

애니메이션 블루프린트의 \*\*'스레드 안전 함수(Thread Safe Functions)'\*\*는 멀티 스레드 업데이트 도중에도 안전하게 호출될 수 있는 함수들입니다.

* **Property Access (속도 최적화):** 언리얼 엔진 5에서는 변수를 직접 가져오는 대신 'Property Access' 기능을 사용하여 게임 스레드의 데이터를 워커 스레드로 안전하게 복사해 옵니다. 이 과정이 Thread Safe하게 이루어집니다.  
* **업데이트 사이클:** BlueprintThreadSafeUpdateAnimation과 같은 함수를 사용하여 매 프레임 애니메이션 로직을 메인 스레드 방해 없이 계산할 수 있습니다.

### **② 제약 사항 (중요)**

함수가 Thread Safe로 설정되면 다음과 같은 작업을 할 수 없습니다.

* **Non-Thread Safe 함수 호출 불가:** Print String, Spawn Actor, Destroy Actor 같이 메인 게임 스레드 전용인 노드들은 사용할 수 없습니다.  
* **일반 액터 변수 직접 참조 제한:** 다른 일반 블루프린트(액터 등)에 선언된 변수를 직접 가져오는 것은 위험하므로, 데이터가 안전하게 복사된 상태여야 합니다.

## **3\. 왜 Thread Safe를 신경 써야 하나요?**

1. **성능 최적화 (Optimization):** 모든 로직을 메인 스레드에서만 처리하면 CPU 코어 하나만 혹사당하게 됩니다. Thread Safe한 설계를 통해 여러 코어를 동시에 활용할 수 있습니다.  
2. **안정성 (Stability):** 스레드 경합(Race Condition)으로 인한 불규칙적인 크래시를 방지합니다.  
3. **병렬 처리:** 특히 수많은 적 캐릭터나 NPC가 등장하는 게임에서 애니메이션 계산 부하를 획기적으로 줄여줍니다.

## **4\. 실무에서의 설정 방법**

1. **함수 생성:** 애니메이션 블루프린트에서 함수를 새로 만듭니다.  
2. **디테일 패널 설정:** 함수의 'Details' 패널에서 **Thread Safe** 체크박스를 활성화합니다.  
3. **로직 구현:** 이때 사용 가능한 노드들이 제한됩니다. 순수(Pure) 함수이거나 다른 Thread Safe 함수들만 연결할 수 있게 됩니다.

## **요약**

언리얼 블루프린트에서 **Thread Safe**는 \*\*"이 로직은 메인 스레드가 아닌 백그라운드 스레드에서 동시에 돌아가도 안전하다"\*\*는 것을 보증하는 설정입니다. 주로 애니메이션 성능을 높이기 위해 CPU의 멀티 코어를 제대로 활용하려는 목적으로 사용됩니다.
