# **Truncate 노드 상세 가이드**

언리얼 엔진 블루프린트의 **Truncate** 노드는 수학적 연산에서 매우 자주 사용되는 형변환 및 절삭 도구입니다.

## **1\. 주요 역할**

* **소수점 제거:** 입력된 Float 또는 Double 값에서 소수점 뒷부분을 완전히 삭제하고 정수 부분만 남깁니다.  
* **형변환:** 주로 Float 타입을 Integer 타입으로 출력할 때 사용됩니다.

## **2\. 작동 방식 (예시)**

Truncate는 양수와 음수에서 각각 다음과 같이 작동합니다.

| 입력 값 (Float) | 출력 값 (Integer) | 특징 |
| :---- | :---- | :---- |
| **2.1** | **2** | 소수점 .1 버림 |
| **2.9** | **2** | .9가 크더라도 반올림하지 않고 버림 |
| **\-2.1** | **\-2** | 음수에서도 소수점 방향을 버림 |
| **\-2.9** | **\-2** | 0에 가까운 쪽 정수로 결과가 나옴 |

## **3\. 다른 노드와의 비교**

Truncate를 정확히 이해하려면 유사한 기능을 하는 다른 노드들과 비교하는 것이 좋습니다.

* **Truncate (절삭):** 무조건 소수점을 버립니다. (0을 향해 이동)  
* **Floor (내림):** 값보다 작거나 같은 가장 큰 정수를 반환합니다. (음수 방향으로 이동)  
* **Ceil (올림):** 값보다 크거나 같은 가장 작은 정수를 반환합니다. (양수 방향으로 이동)  
* **Round (반올림):** 소수점이 .5 이상이면 올리고, 미만이면 내립니다.

### **비교 예시**

| 입력값 | Truncate | Floor | Ceil | Round |
| :---- | :---- | :---- | :---- | :---- |
| **2.7** | 2 | 2 | 3 | 3 |
| **\-2.7** | \-2 | \-3 | \-2 | \-3 |

## **4\. 주로 사용되는 상황**

1. **배열 인덱스 계산:** 계산된 위치 값을 배열의 인덱스(정수)로 변환해야 할 때.  
2. **UI 표시:** 체력(HP)이나 점수 등을 소수점 없이 정수로만 화면에 보여주고 싶을 때.  
3. **그리드 시스템:** 캐릭터의 위치를 특정 칸(Grid) 단위의 정수 좌표로 고정할 때.

## **5\. 팁**

음수 계산이 포함된 로직에서는 **Truncate**와 **Floor**의 결과가 다르다는 점에 주의해야 합니다. 단순히 "숫자만 떼어내고 싶다"면 Truncate가 적합하고, "항상 아래 단계로 내리고 싶다"면 Floor를 사용해야 합니다.
