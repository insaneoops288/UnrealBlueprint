# **Unreal Engine: Unrotate Vector 노드 상세 가이드**

Unrotate Vector 노드는 벡터 연산에서 매우 중요한 역할을 하며, 특히 \*\*월드 공간(World Space)\*\*의 데이터를 \*\*로컬 공간(Local Space)\*\*으로 변환할 때 필수적으로 사용됩니다.

## **1\. 핵심 개념**

이 노드는 입력받은 \*\*Rotation(회전값)\*\*의 \*\*반대 방향(Inverse)\*\*으로 \*\*V(벡터)\*\*를 회전시킵니다.

* **Rotate Vector**: 벡터를 해당 회전만큼 돌립니다. (로컬 \-\> 월드)  
* **Unrotate Vector**: 벡터를 해당 회전의 반대만큼 돌립니다. (월드 \-\> 로컬)

## **2\. 작동 원리**

수학적으로는 입력된 Rotator를 역행렬(Inverse) 또는 역쿼터니언으로 변환한 뒤 벡터와 곱하는 것과 같습니다.

* **입력 A (V)**: 회전시키고자 하는 대상 벡터 (예: 월드 좌표계의 속도).  
* **입력 B (B)**: 기준이 되는 회전값 (예: 캐릭터의 현재 회전값).  
* **출력**: 역회전이 적용된 결과 벡터.

## **3\. 주요 활용 사례**

### **① 월드 속도를 로컬 속도로 변환**

캐릭터가 월드상에서 어느 방향으로 움직이든, 캐릭터의 정면을 기준으로 "앞으로 가고 있는지, 옆으로 가고 있는지" 판별할 때 사용합니다.

* **V**: 캐릭터의 Get Velocity  
* **B**: 캐릭터의 Get Actor Rotation  
* **결과**: X가 양수면 전진, Y가 양수면 오른쪽 이동 중임을 의미합니다. (애니메이션 블루프린트의 블렌드 스페이스 파라미터로 자주 사용됨)

### **② 상대적인 위치 계산**

특정 액터(A)를 기준으로 다른 액터(B)가 어느 방향에 있는지 계산할 때 유용합니다.

1. B의 위치 \- A의 위치를 계산하여 방향 벡터를 얻습니다.  
2. 이 벡터를 Unrotate Vector에 넣고 A의 회전값을 연결합니다.  
3. 결과값의 X, Y, Z를 통해 B가 A의 앞/뒤, 좌/우, 위/아래 중 어디에 있는지 정확히 알 수 있습니다.

## **4\. Rotate Vector vs Unrotate Vector 비교**

| 특징 | Rotate Vector | Unrotate Vector |
| :---- | :---- | :---- |
| **방향** | 로컬 → 월드 (순방향) | 월드 → 로컬 (역방향) |
| **주 용도** | 캐릭터 정면 방향으로 장풍 쏘기 | 캐릭터 기준 피격 방향 판별하기 |
| **수학적 의미** | 벡터 \* 회전값 | 벡터 \* (회전값의 역) |

## **5\. 팁**

* 만약 결과값이 예상과 반대로 나온다면, Rotate Vector를 써야 할 상황에 Unrotate Vector를 쓴 것은 아닌지 확인해 보세요.  
* 단순히 방향만 필요한 경우(크기가 1인 벡터), 입력 벡터를 미리 Normalize 하는 것이 계산상 안전합니다.
