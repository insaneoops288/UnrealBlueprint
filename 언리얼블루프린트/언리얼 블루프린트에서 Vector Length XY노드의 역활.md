언리얼 블루프린트에서 Vector Length XY노드의 역활?

언리얼 엔진 블루프린트에서 Vector Length XY 노드는 입력된 벡터의 X와 Y축 값만을 사용하여 해당 벡터의 길이를 계산하는 역할을 합니다. 즉, 벡터의 수직 성분인 Z축 값은 완전히 무시합니다.

1. 주요 기능 및 수학적 원리
이 노드는 3차원 벡터(X,Y,Z)를 입력받지만, 내부적으로는 2차원 평면상의 길이를 구하는 공식을 사용합니다.

 
Vector Length 노드와의 차이: 일반적인 Vector Length 노드가 $\sqrt{X^2 + Y^2 + Z^2}$를 계산하는 것과 달리, 이 노드는 Z값을 0으로 간주하고 계산하는 것과 같습니다.

2. 언제 주로 사용하나요?
가장 대표적인 활용 사례는 "수평 거리" 또는 **"지면 속도"**를 측정할 때입니다.

캐릭터의 이동 속도(Ground Speed) 계산: 캐릭터가 점프 중이거나 경사로를 내려가고 있을 때, 순수하게 앞/뒤/왼쪽/오른쪽으로 움직이는 속도만 체크하고 싶을 때 사용합니다. Get Velocity 노드의 결과값을 이 노드에 연결하면 수직 낙하 속도를 제외한 이동 속도를 얻을 수 있습니다.

평면상의 거리 측정: 두 액터 사이의 거리를 잴 때, 높이 차이는 무시하고 지도상의 직선 거리(Top-down 거리)만 필요할 경우 두 위치의 차(Subtract) 벡터를 이 노드에 넣습니다.

UI 및 미니맵: 3D 공간의 위치를 2D 미니맵 좌표로 변환하거나, 평면적인 방향 벡터의 세기를 조절할 때 유용합니다.

3. 노드 구성 요약
항목	설명
Input (V)	길이를 측정할 Vector (3D 벡터)
Output (Return Value)	X, Y 값으로만 계산된 Float (길이 값)
요약하자면: 높이(Z)를 제외한 평면적인 움직임이나 거리를 파악하고 싶을 때 Vector Length 대신 Vector Length XY를 사용하면 됩니다.
