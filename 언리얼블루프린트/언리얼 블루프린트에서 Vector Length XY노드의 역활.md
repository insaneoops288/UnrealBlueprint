언리얼 블루프린트에서 **Vector Length XY** 노드는 입력받은 벡터의 $X$와 $Y$ 성분만을 사용하여 그 \*\*길이(크기)\*\*를 계산하는 노드입니다.

쉽게 말해, 3D 공간 상의 벡터에서 \*\*Z축(높이) 값을 완전히 무시하고 "바닥 평면상의 길이"\*\*만 뽑아내는 역할을 합니다.

## ---

**1\. 수학적 정의**

이 노드는 피타고라스의 정리를 사용하여 다음과 같은 연산을 수행합니다.

$$\\text{Length XY} \= \\sqrt{X^2 \+ Y^2}$$  
일반적인 **Vector Length** 노드가 $\\sqrt{X^2 \+ Y^2 \+ Z^2}$를 계산하는 것과 달리, $Z$ 값을 계산에서 제외하기 때문에 오직 수평적인 크기만 결과로 나옵니다.

## ---

**2\. 왜 사용하나요? (주요 활용 사례)**

### **① 캐릭터의 "이동 속도(Ground Speed)" 계산**

가장 많이 쓰이는 용도입니다. 애니메이션 블루프린트에서 캐릭터가 걷거나 뛰는 속도를 체크할 때 필수적입니다.

* **문제:** 캐릭터가 점프하거나 떨어질 때 Get Velocity의 길이를 그냥 재면, 아래위로 움직이는 속도까지 포함되어 'Speed' 값이 튀게 됩니다.  
* **해결:** Vector Length XY를 사용하면 캐릭터가 공중에 떠 있더라도 **수평으로 이동하는 순수한 달리기 속도**만 정확히 측정할 수 있습니다.

### **② 평면 거리 측정**

두 오브젝트가 서로 다른 높이(예: 빌딩 위와 아래)에 있을 때, 높이 차이를 무시하고 \*\*"지도상에서 얼마나 멀리 떨어져 있는가"\*\*를 알고 싶을 때 사용합니다.

* 두 위치를 뺀(-) 결과 벡터를 이 노드에 넣으면 수평 거리만 나옵니다.

### **③ 발사체 전진 힘 계산**

투척물이나 총알이 위아래 곡선을 그리며 날아가더라도, 수평 방향으로만 가해지는 힘의 크기를 일정하게 유지하거나 체크하고 싶을 때 유용합니다.

## ---

**3\. Vector Length vs Vector Length XY 비교**

| 노드 이름 | 대상 성분 | 용도 |
| :---- | :---- | :---- |
| **Vector Length** | X, Y, Z | 실제 3D 공간에서의 물리적 속력, 전체 거리 |
| **Vector Length XY** | **X, Y** | **수평 이동 속도, 지도상의 거리, 레이더 상의 위치** |

## ---

**💡 실무 팁: "애니메이션 블루프린트 필수 공식"**

애니메이션 블루프린트의 Update Animation 이벤트에서 캐릭터의 **Speed** 변수를 업데이트할 때 다음과 같이 구성하는 것이 정석입니다.

1. Try Get Pawn Owner → Get Velocity  
2. Get Velocity → **Vector Length XY**  
3. 결과값을 **Speed** 변수에 저장

이렇게 하면 캐릭터가 점프 중이든, 사다리를 타고 있든 바닥에서의 이동 속도에 맞춰 애니메이션 블렌드 스페이스가 정확하게 작동합니다.

**요약**

**Vector Length XY**는 "높낮이(Z)는 상관없고, \*\*평면 위에서 얼마나 빠른지(혹은 먼지)\*\*만 알려줘\!"라고 할 때 쓰는 노드입니다.

혹시 지금 캐릭터의 달리기 애니메이션이 점프할 때 어색하게 변하는 문제를 해결하려고 하시나요? 이 노드를 적용하면 즉시 해결될 거예요\! 더 궁금한 점이 있으신가요?
