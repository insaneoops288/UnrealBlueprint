# **Event RadialDamage의 역할과 활용**

언리얼 엔진에서 Event RadialDamage는 특정 지점을 중심으로 반경 내에 있는 액터들에게 데미지를 전달할 때 호출되는 이벤트입니다. 주로 **수류탄 폭발, 포격, 충격파** 등을 구현할 때 핵심적인 역할을 합니다.

## **1\. 주요 역할**

이 이벤트는 액터가 '범위 피해'를 입었다는 사실을 알리고, 그 과정에서 필요한 구체적인 정보(데미지 양, 폭발 중심지 등)를 제공합니다. 개발자는 이 이벤트를 통해 캐릭터의 체력을 깎거나, 폭발 방향으로 밀려나게 하는(Knockback) 등의 로직을 실행할 수 있습니다.

## **2\. 주요 파라미터 (입력 핀) 설명**

Event RadialDamage가 발생하면 다음과 같은 유용한 정보들을 얻을 수 있습니다.

| 파라미터 | 설명 |
| :---- | :---- |
| **Damage Amount** | 액터가 실제로 받은 데미지 수치입니다. |
| **Damage Type** | 데미지의 성격(예: 화염, 전기, 물리 등)을 정의하는 클래스입니다. |
| **Origin** | 폭발이나 충격이 시작된 중심 위치(Vector)입니다. |
| **Hit Info** | 충격이 가해진 구체적인 지점, 노멀(방향) 등의 레이캐스트 결과 정보입니다. |
| **Instigated By** | 데미지를 유발한 원인 제공자(보통 Controller)입니다. |
| **Damage Causer** | 데미지를 직접적으로 입힌 액터(예: 수류탄 액터 자체)입니다. |

## **3\. 다른 데미지 이벤트와의 차이점**

언리얼에는 크게 세 가지 데미지 이벤트가 있습니다.

1. **Event AnyDamage**: 모든 종류의 데미지에 반응하는 가장 포괄적인 이벤트입니다.  
2. **Event PointDamage**: 총기 사격처럼 특정 지점(Point)을 타격했을 때 발생하며, 피격 부위나 탄환 궤적 정보를 포함합니다.  
3. **Event RadialDamage**: \*\*중심점(Origin)\*\*이 존재하며, 중심에서 멀어질수록 데미지가 감소하는 **감쇠(Falloff)** 로직을 기본적으로 지원합니다.

## **4\. 작동 원리 (Flow)**

1. **발생**: 공격자 액터에서 Apply Radial Damage 또는 Apply Radial Damage with Falloff 함수를 호출합니다.  
2. **판정**: 엔진이 지정된 반경 내에 있는 액터들을 찾습니다. (필요 시 장애물에 가려졌는지 체크하는 라인 트레이스 수행)  
3. **호출**: 피해 범위 안에 있는 액터들의 블루프린트에서 Event RadialDamage가 실행됩니다.

## **5\. 활용 팁: 넉백(Knockback) 구현**

RadialDamage의 Origin 핀을 활용하면 아주 쉽게 넉백 효과를 만들 수 있습니다.

* **로직**: 액터의 현재 위치 \- Origin을 계산하여 방향 벡터를 구한 뒤, 그 방향으로 Launch Character나 Add Impulse를 실행하면 폭발 중심에서 바깥쪽으로 날아가는 효과를 낼 수 있습니다.

### **요약**

Event RadialDamage는 단순히 "아프다"는 정보뿐만 아니라 \*\*"어디서 터졌고, 어느 정도의 충격이 전달되었는가"\*\*를 알려주는 이벤트입니다. 폭발물이나 범위 공격이 있는 게임을 만든다면 반드시 이해해야 하는 중요한 기능입니다.
