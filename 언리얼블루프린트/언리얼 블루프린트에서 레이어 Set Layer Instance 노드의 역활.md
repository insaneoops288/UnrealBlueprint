# **언리얼 엔진: 애니메이션 레이어 인스턴스 설정 (Set Layer Instance) 가이드**

언리얼 엔진의 애니메이션 블루프린트(AnimBP) 시스템에서 \*\*애니메이션 레이어(Animation Layers)\*\*는 복잡한 애니메이션 로직을 모듈화하고 재사용할 수 있게 해주는 강력한 기능입니다. 이 과정에서 Set Layer Instance 또는 Link Anim Class Layers 노드는 런타임에 애니메이션 로직을 동적으로 교체하는 스위치 역할을 합니다.

## **1\. 노드의 핵심 역할**

**Set Layer Instance**의 주된 역할은 \*\*"특정 레이어 인터페이스에 정의된 슬롯에 실제 동작할 애니메이션 블루프린트 클래스를 할당하는 것"\*\*입니다.

* **동적 교체**: 게임 실행 중에 캐릭터의 상태(예: 무기 장착, 부상 상태 등)에 따라 애니메이션 로직을 즉시 바꿀 수 있습니다.  
* **모듈화**: 하나의 거대한 AnimBP를 만드는 대신, '걷기 레이어', '전투 레이어', '아이들 레이어' 등으로 분리하여 관리할 수 있게 합니다.  
* **메모리 효율성**: 필요한 상황에만 특정 레이어 클래스를 로드하고 연결하여 메모리를 최적화할 수 있습니다.

## **2\. 작동 원리 및 구조**

이 기능을 사용하기 위해서는 보통 세 가지 요소가 필요합니다.

### **A. 애니메이션 레이어 인터페이스 (Animation Layer Interface)**

애니메이션 함수들의 "틀"을 정의합니다. 예를 들어 UpperBodyLayer라는 함수가 포함된 인터페이스를 만듭니다.

### **B. 메인 애니메이션 블루프린트 (Main AnimBP)**

이 인터페이스를 상속받으며, 애님 그래프 내에서 **Linked Anim Layer** 노드를 사용하여 레이어가 들어갈 "자리"를 마련해둡니다.

### **C. 레이어 전용 애니메이션 블루프린트 (Layer AnimBP)**

동일한 인터페이스를 상속받아 실제로 UpperBodyLayer 안에서 어떤 애니메이션(예: 소총 조준, 권총 조준)이 재생될지 구현합니다.

## **3\. 노드 사용 예시 (Blueprint)**

주로 캐릭터 블루프린트나 컴포넌트에서 다음과 같은 흐름으로 사용됩니다.

1. **Target**: 레이어를 설정할 대상 애니메이션 인스턴스(Mesh \-\> Get Anim Instance).  
2. **In Class**: 실행 시점에 연결하고자 하는 **레이어 전용 애니메이션 블루프린트 클래스**.  
3. **결과**: 호출 즉시 메인 AnimBP의 해당 레이어 슬롯이 새로운 클래스의 로직으로 대체됩니다.

## **4\. 실전 활용 사례**

| 상황 | 레이어 활용 방식 |
| :---- | :---- |
| **무기 시스템** | 소총을 들면 RifleAnimLayer를, 검을 들면 SwordAnimLayer를 연결합니다. |
| **탈것/탑승** | 말을 타거나 차에 탈 때 상체 레이어만 탑승용 애니메이션 클래스로 교체합니다. |
| **부상 시스템** | 캐릭터의 체력이 낮아지면 InjuredMovementLayer를 연결해 걷는 방식을 바꿉니다. |
| **다양한 직업군** | 기본 이동 로직은 같지만 캐릭터 직업(전사, 마법사)에 따라 대기 동작 레이어만 다르게 설정합니다. |

## **5\. 주의사항 및 팁**

* **인터페이스 일치**: Link Anim Class Layers 노드를 사용할 때는 연결하려는 클래스가 메인 AnimBP가 사용하는 인터페이스를 반드시 구현하고 있어야 합니다.  
* **상태 보존**: 레이어를 교체할 때 이전 레이어의 변수 값은 유지되지 않으므로, 필요한 데이터는 메인 AnimBP나 캐릭터 블루프린트에서 전달해야 합니다.  
* **성능**: 매 프레임 호출할 필요가 없습니다. 무기가 바뀌거나 상태가 변하는 **이벤트 시점**에 한 번만 호출하면 됩니다.

### **요약**

Set Layer Instance는 **애니메이션의 부품(레이어)을 실시간으로 갈아끼우는 노드**입니다. 이를 통해 개발자는 확장성이 뛰어난 캐릭터 애니메이션 시스템을 구축할 수 있습니다.
