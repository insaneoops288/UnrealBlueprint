# **Unreal Engine: Get Up Vector 노드 이해하기**

Get Up Vector 노드는 액터나 컴포넌트의 \*\*로컬 Z축(위쪽)\*\*이 세계 좌표계(World Space)에서 어떤 방향인지를 나타내는 \*\*단위 벡터(Unit Vector)\*\*를 반환합니다.

## **1\. 핵심 개념**

* **로컬 vs 월드**: 모든 액터는 자신만의 앞(X), 오른쪽(Y), 위(Z) 축을 가집니다. 액터가 회전하면 이 축들도 함께 회전합니다. Get Up Vector는 이 회전된 상태의 '위쪽' 방향이 전체 게임 세상(월드) 기준으로는 어느 좌표 방향인지를 계산해서 알려줍니다.  
* **정규화(Normalized)**: 반환되는 벡터의 길이는 항상 **1.0**입니다. 이는 방향 정보만을 담고 있음을 의미합니다.

## **2\. 주요 특징**

* **좌표계**: 언리얼 엔진은 **Z-Up** 오른손 좌표계를 사용합니다. 기본 상태(회전값 0, 0, 0)에서 Get Up Vector 값은 (0, 0, 1)입니다.  
* **회전 대응**: 액터가 옆으로 90도 기울어지면, 위쪽 벡터는 월드의 수평 방향을 가리키게 됩니다.

## **3\. 주요 활용 사례**

### **A. 캐릭터 이동 및 발사체**

* **공중 부양/점프**: 캐릭터가 단순히 월드 좌표의 위쪽(0, 0, 1)이 아니라, 자신의 머리 방향으로 솟구치게 하고 싶을 때 사용합니다.  
* **부스트 효과**: 비행기나 우측으로 기울어진 발사체가 자신의 상단 방향으로 추진력을 얻을 때 Get Up Vector \* 힘(Float) 연산을 통해 힘을 가합니다.

### **B. 소환 및 배치**

* **아이템 장착**: 캐릭터의 머리 위에 말풍선이나 이펙트를 띄울 때, 캐릭터의 위치에 Get Up Vector \* 거리를 더해 정확한 '머리 위' 좌표를 구합니다.  
* **표면 정렬**: 레이캐스트(Line Trace)를 통해 바닥의 기울기(Normal)를 얻은 후, 액터의 Up Vector를 그 노멀값에 맞추면 경사면에 자연스럽게 서 있는 모습을 연출할 수 있습니다.

### **C. 상태 판정**

* **뒤집힘 체크**: 차가 뒤집혔는지 확인하고 싶을 때, Get Up Vector와 월드의 위쪽 방향 (0, 0, 1)을 **Dot Product(내적)** 합니다. 결과값이 마이너스(-)라면 차가 뒤집힌 상태라고 판단할 수 있습니다.

## **4\. 관련 노드들**

함께 자주 쓰이는 노드들은 다음과 같습니다:

* **Get Forward Vector**: 액터의 정면(X축) 방향을 가져옵니다.  
* **Get Right Vector**: 액터의 오른쪽(Y축) 방향을 가져옵니다.  
* **Get Actor Rotation**: 현재 액터의 회전값 자체를 가져올 때 사용합니다.

### **요약**

Get Up Vector는 \*\*"이 물체의 입장에서 '위'는 지금 세상 어디인가?"\*\*라는 질문에 대한 답을 주는 노드입니다.
