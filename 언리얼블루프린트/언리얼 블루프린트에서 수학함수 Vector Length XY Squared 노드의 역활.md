# **Vector Length XY Squared 노드 가이드**

언리얼 엔진 블루프린트에서 Vector Length XY Squared 노드는 특정 벡터의 ![][image1]축 값을 무시하고, ![][image2] 평면상에서의 길이를 **제곱된 형태**로 반환합니다.

## **1\. 수학적 정의**

이 노드가 수행하는 계산식은 다음과 같습니다:

![][image3]일반적인 Vector Length XY 노드가 $\\sqrt{X^2 \+ Y^2}$를 계산하는 것과 달리, 마지막 단계인 **제곱근(Square Root) 계산을 생략**합니다.

## **2\. 주요 특징**

* **Z축 무시:** 3D 공간의 벡터 입력을 받지만, 높이(![][image1]) 데이터는 계산에 포함하지 않습니다. 이는 캐릭터가 지면 위에서 얼마나 멀리 있는지 등 2D 거리를 측정할 때 유용합니다.  
* **성능 최적화:** 컴퓨터 그래픽스 연산에서 제곱근(![][image4]) 연산은 상대적으로 비용이 많이 드는 작업입니다. 수백 명의 AI 거리를 매 프레임마다 체크해야 한다면, 제곱근을 뺀 이 노드를 사용하는 것이 성능 면에서 훨씬 유리합니다.

## **3\. 언제 사용하는가?**

### **① 거리 비교 (Distance Comparison)**

두 대상 사이의 실제 거리가 필요한 것이 아니라, 단순히 \*\*"A가 B보다 더 먼가?"\*\*를 판단할 때 가장 많이 사용됩니다.

* **예시:** Vector Length XY Squared 값이 ![][image5](![][image6])보다 큰지 확인하는 것이, Vector Length XY 값이 ![][image7]보다 큰지 확인하는 것보다 빠릅니다.

### **② 성능 최적화 (Optimization)**

* 수많은 액터의 거리를 실시간으로 체크해야 하는 군중 시스템이나 탄막 게임 등에서 일반 Length 대신 사용합니다.

### **③ 특정 반경 체크**

* 캐릭터가 특정 원형 영역(Cylinder 형태의 범위) 안에 들어왔는지 확인할 때, 높낮이에 상관없이 ![][image2] 평면상의 거리만 체크하고 싶을 때 사용합니다.

## **4\. 노드 비교**

| 노드 이름 | 공식 | 특징 |
| :---- | :---- | :---- |
| **Vector Length** | **![][image8]** | 3D 전체 공간의 실제 거리 |
| **Vector Length XY** | **![][image9]** | 평면상의 실제 거리 (![][image1] 무시) |
| **Vector Length Squared** | **![][image10]** | 3D 거리의 제곱 (최적화용) |
| **Vector Length XY Squared** | **![][image11]** | **평면 거리의 제곱 (최적화용, ![][image1] 무시)** |

## **5\. 블루프린트 활용 팁**

만약 특정 거리 MaxDistance 내에 있는지 확인하고 싶다면 아래와 같이 로직을 구성하세요:

1. **Target Vector**에서 **Self Location**을 빼서 상대 벡터를 구합니다.  
2. Vector Length XY Squared 노드에 연결합니다.  
3. 비교할 거리인 MaxDistance를 **제곱**(MaxDistance \* MaxDistance) 합니다.  
4. 두 값을 비교(\> 또는 \<) 합니다.
