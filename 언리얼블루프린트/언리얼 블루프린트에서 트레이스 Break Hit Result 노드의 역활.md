# **Break Hit Result 노드의 역할과 주요 핀(Pin) 설명**

언리얼 엔진에서 Line Trace By Channel과 같은 노드를 실행하면 출력 핀으로 Out Hit이 나옵니다. 이 핀을 끌어다 Break Hit Result 노드를 연결하면 충돌에 대한 모든 세부 정보를 확인할 수 있습니다.

## **1\. 주요 출력 핀 상세 설명**

| 핀 이름 | 데이터 타입 | 설명 |
| :---- | :---- | :---- |
| **Blocking Hit** | Boolean | 트레이스가 무언가(차단 설정된 물체)에 부딪혔는지 여부를 반환합니다. |
| **Hit Actor** | Actor Object | 충돌한 대상 액터가 무엇인지 알려줍니다. (예: Apply Damage의 대상 지정 시 사용) |
| **Hit Component** | Primitive Component | 액터 내의 구체적으로 어떤 컴포넌트(메시, 콜리전 등)에 맞았는지 알려줍니다. |
| **Location** | Vector | 트레이스가 끝난 지점 또는 충돌 지점의 좌표입니다. |
| **Impact Point** | Vector | 실제 표면과 접촉한 정확한 지점입니다. (이펙트 스폰 시 주로 사용) |
| **Normal** | Vector | 충돌한 표면의 \*\*법선 벡터(수직 방향)\*\*입니다. 캐릭터가 벽을 타고 흐르거나 탄환이 튕겨 나가는 계산을 할 때 필수적입니다. |
| **Phys Mat** | Physical Material | 충돌한 바닥이나 벽의 물리 재질입니다. (발소리 사운드나 피격 데칼을 바꿀 때 사용) |
| **Distance** | Float | 트레이스 시작점부터 충돌 지점까지의 거리입니다. |
| **Bone Name** | Name | 스켈레탈 메시일 경우, 정확히 어떤 뼈(Bone)에 맞았는지 반환합니다. (헤드샷 판정 등에 활용) |

## **2\. 실전 활용 사례**

### **1\) 피격 이펙트 및 데칼 생성**

총기 발사를 구현할 때 Impact Point는 이펙트가 생성될 위치가 되고, Normal은 이펙트나 데칼이 표면에 수직으로 정렬되도록 회전값(Rotation From X Vector)을 만드는 데 사용됩니다.

### **2\) 재질에 따른 차별화된 처리**

Phys Mat을 사용하여 바닥이 '금속'이면 챙그랑 소리를, '흙'이면 퍽 하는 소리를 재생하도록 분기 처리(Switch on Physical Surface)를 할 수 있습니다.

### **3\) 대미지 전달**

Hit Actor를 Apply Damage 노드에 연결하여 내가 맞춘 대상에게만 정확히 대미지를 입힐 수 있습니다.

## **3\. 팁: 노드 확장하기**

Break Hit Result 노드는 기본적으로 몇 개의 핀만 보일 수 있습니다. 노드를 선택하고 우측 **디테일(Details) 패널**에서 확인하고 싶은 항목의 체크박스를 켜거나, 노드 하단의 **화살표 모양 아이콘**을 클릭하면 숨겨진 모든 핀을 확장해서 볼 수 있습니다.
