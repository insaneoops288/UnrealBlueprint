# **Sphere Trace (구체 트레이스) 상세 가이드**

## **1\. Sphere Trace의 핵심 역할**

Sphere Trace는 시작점(Start)에서 끝점(End)까지 지정된 \*\*반지름(Radius)\*\*을 가진 구체를 이동시키며 경로상에 있는 콜리전(Collision)을 감지합니다.

* **범위 감지:** 직선이 아니라 일정한 '부피'를 가진 영역을 체크합니다.  
* **정밀도 향상:** 아주 가느다란 Line Trace로 놓칠 수 있는 작은 물체나 좁은 틈새를 확실하게 잡아냅니다.

## **2\. 주요 입력 파라미터 (Inputs)**

노드에서 설정해야 하는 주요 항목들은 다음과 같습니다.

| 항목 | 설명 |
| :---- | :---- |
| **Start** | 트레이스가 시작되는 3D 공간의 위치 (Vector). |
| **End** | 트레이스가 끝나는 3D 공간의 위치 (Vector). |
| **Radius** | 감지할 구체의 크기(반지름). 값이 클수록 더 넓은 범위를 체크합니다. |
| **Trace Channel** | 어떤 콜리전 채널(Visibility, Camera 등)을 대상으로 할지 결정합니다. |
| **Actors to Ignore** | 감지 대상에서 제외할 액터 목록 (보통 자기 자신을 넣습니다). |
| **Draw Debug Type** | 화면에 트레이스 경로를 가시적으로 보여줍니다. (None, One Frame, For Duration, Persistent) |

## **3\. 출력 데이터 (Outputs)**

트레이스가 완료되면 다음과 같은 정보를 반환합니다.

* **Return Value (Boolean):** 무언가에 맞았다면 True, 아무것도 맞지 않았다면 False를 반환합니다.  
* **Out Hit (Hit Result Struct):** 충돌한 대상에 대한 상세 정보를 담고 있습니다. Break Hit Result 노드를 연결하여 다음 정보를 추출할 수 있습니다.  
  * **Hit Actor:** 맞은 대상이 누구인지.  
  * **Location:** 구체가 물체와 처음 부딪힌 지점.  
  * **Normal:** 부딪힌 표면의 방향(법선 벡터).  
  * **Distance:** 시작점에서 충돌 지점까지의 거리.

## **4\. Line Trace vs Sphere Trace 비교**

| 특징 | Line Trace | Sphere Trace |
| :---- | :---- | :---- |
| **형태** | 아주 가느다란 선 | 부피가 있는 공 모양 |
| **용도** | 정확한 조준 사격, 클릭 감지 | 근접 공격 판정, 주변 장애물 감지 |
| **정밀도** | 너무 가늘어서 작은 물체를 지나칠 수 있음 | 설정한 반지름 덕분에 안정적으로 감지 |

## **5\. 실전 활용 사례**

### **① 근접 공격 판정 (Melee Combat)**

칼을 휘두를 때 칼날의 궤적을 따라 Sphere Trace를 실행하면, 얇은 선보다 훨씬 자연스럽게 적을 타격할 수 있습니다.

### **② 캐릭터 바닥 체크 (Ground Check)**

캐릭터가 경사로나 울퉁불퉁한 지형에 서 있을 때, 발 밑에 구체 트레이스를 쏘면 캐릭터의 캡슐 콜리전보다 더 부드럽게 지면 높이를 계산할 수 있습니다.

### **③ 상호작용 (Interaction)**

플레이어 앞에 아이템이 있는지 확인할 때, 정확히 아이템의 중심을 조준하지 않아도 근처에 구체가 닿으면 상호작용 팝업을 띄우는 용도로 사용합니다.

### **④ 장애물 회피 (Avoidance)**

AI나 카메라가 이동 경로상에 벽이 있는지 미리 파악하여 부드럽게 경로를 수정할 때 사용합니다.

## **6\. 사용 시 팁**

* **성능:** Line Trace보다 연산량이 미세하게 많지만, 현대의 하드웨어에서는 수십 개를 동시에 써도 큰 무리가 없습니다.  
* **디버깅:** 처음 구현할 때는 Draw Debug Type을 **For Duration**으로 설정하여 구체가 실제로 어디를 훑고 지나가는지 눈으로 확인하는 것이 필수적입니다.
