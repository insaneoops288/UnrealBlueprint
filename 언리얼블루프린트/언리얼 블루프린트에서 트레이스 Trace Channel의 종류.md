# **언리얼 엔진 트레이스(Trace) 채널 및 종류 가이드**

트레이스는 게임 월드에서 "어떤 물체가 앞에 있는가?" 또는 "시야가 확보되었는가?" 등을 판단할 때 사용됩니다.

## **1\. 트레이스 방식에 따른 분류**

### **① Trace by Channel (채널 기반 트레이스)**

특정 **트레이스 채널**에 어떻게 반응하도록 설정되었는지를 기준으로 감지합니다. 가장 흔히 사용되는 방식입니다.

* **Visibility (가시성):** 기본적으로 "눈에 보이는가"를 판단합니다. 레이저 포인터, 클릭 피킹(Picking) 등에 주로 사용됩니다.  
* **Camera (카메라):** 카메라 시야와 캐릭터 사이에 장애물이 있는지 판단합니다. 카메라가 벽을 뚫지 않게 하거나 시야를 가리는 물체를 찾을 때 사용됩니다.

### **② Trace for Objects (오브젝트 기반 트레이스)**

물체의 \*\*오브젝트 유형(Object Type)\*\*을 기준으로 감지합니다. 채널 반응(Block/Overlap)과 상관없이 해당 타입의 물체면 무조건 검출하고 싶을 때 유용합니다.

* **WorldStatic:** 움직이지 않는 배경 (바닥, 벽 등).  
* **WorldDynamic:** 움직이는 액터 (문, 움직이는 발판 등).  
* **Pawn:** 플레이어 캐릭터나 AI.  
* **PhysicsBody:** 물리 시뮬레이션이 적용된 물체.  
* **Vehicle:** 차량 오브젝트.

## **2\. 형태(Shape)에 따른 분류**

단순한 선(Line) 외에도 다양한 도형을 날려 범위를 체크할 수 있습니다.

| 종류 | 특징 | 주요 용도 |
| :---- | :---- | :---- |
| **Line Trace** | 시작점에서 끝점까지 가느다란 선을 쏩니다. | 총알 발사, 스위치 작동, 시선 체크 |
| **Sphere Trace** | 구체 형태를 특정 경로로 이동시키며 체크합니다. | 근접 공격 판정, 폭발 범위 체크 |
| **Box Trace** | 상자 형태를 경로를 따라 이동시킵니다. | 넓은 통로의 장애물 감지 |
| **Capsule Trace** | 캐릭터 모양인 캡슐 형태로 체크합니다. | 캐릭터가 지나갈 수 있는 공간인지 확인 |

## **3\. 결과 개수에 따른 분류**

* **Single Trace:** 가장 먼저 부딪힌 **하나의 물체** 정보만 반환합니다. (LineTraceSingle...)  
* **Multi Trace:** 경로상에 부딪힌 **모든 물체**를 배열(Array) 형태로 반환합니다. 관통하는 총알이나 범위 내 여러 적을 감지할 때 사용합니다. (LineTraceMulti...)

## **4\. 커스텀 트레이스 채널 만들기**

기본 제공되는 Visibility나 Camera 외에 게임 전용 채널(예: Bullet, Interaction)이 필요할 때 직접 추가할 수 있습니다.

1. **Project Settings (프로젝트 세팅)** 메뉴로 들어갑니다.  
2. **Engine \-\> Collision** 항목을 선택합니다.  
3. **Trace Channels** 섹션에서 New Trace Channel... 버튼을 클릭합니다.  
4. 이름을 입력하고 기본 응답(Default Response)을 설정합니다.  
   * **Ignore:** 기본적으로 무시 (필요한 것만 Block 설정)  
   * **Block:** 기본적으로 충돌 (통과시키고 싶은 것만 Ignore 설정)

## **5\. 트레이스 결과 (Hit Result) 활용**

트레이스가 성공하면 Hit Result 구조체를 통해 다음과 같은 정보를 얻을 수 있습니다.

* **Blocking Hit:** 무언가에 부딪혔는지 여부 (Boolean)  
* **Hit Actor:** 부딪힌 액터가 무엇인지  
* **Location / Impact Point:** 부딪힌 정확한 위치 좌표  
* **Normal / Impact Normal:** 부딪힌 면의 방향 (벡터)  
* **Phys Mat:** 부딪힌 물체의 물리 재질 (발소리 사운드 구분 등에 사용)
